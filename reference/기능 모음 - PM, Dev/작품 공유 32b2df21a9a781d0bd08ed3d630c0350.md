# 작품 공유

최종 편집 일시: 2026년 3월 30일 오후 3:30

## 개요

단건 작품을 공유하는 기능. 전시 초대장 공유와 달리, 작업물 상세 모달에서 단건 작품을 공유하는 기능.

## 진입점

- 작업물 상세 모달 우측 액션 영역 공유 버튼
- 내 프로필 > **작품 관리 탭** > 작품 카드 `...` 메뉴

## 공유 방식

- OS 기본 공유 시트 (Share Sheet) 활용
- 작품 URL + OG 이미지 (전시명 + 대표 이미지 + 작가명)

## 전시 초대장 공유와 다른 점

|  | 작품 공유 | 전시 초대장 공유 |
| --- | --- | --- |
| 대상 | 단건 작품 | 그룹전시 전체 |
| 형식 | URL + OG 이미지 | 전시 초대장 랜딩 페이지 |
| 방명록 | ❌ | ❌ (Phase 2) |

## 미결 사항

- [x]  작품 OG 이미지 구성 규칙 → **전시명 + 대표 이미지 + 작가명** ✅
- [x]  공유 후 작품 상세 URL 동작 → **URL 변경 + 모달 open** ✅ (history.pushState 방식)
