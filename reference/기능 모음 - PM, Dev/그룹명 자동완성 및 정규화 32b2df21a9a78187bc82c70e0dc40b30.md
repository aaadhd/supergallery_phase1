# 그룹명 자동완성 및 정규화

관련 화면: 작업 업로드

> 그룹명 입력 시 오타·띄어쓰기 차이로 다른 그룹으로 분리되는 문제를 방지하는 기능.
> 

---

## 개요

업로드 세부 정보 설정 모달의 그룹명 필드에 적용되는 자동완성 및 정규화 기능.

## 진입점

- 작업 업로드 > 세부 정보 설정 모달 > 그룹명 입력 필드

## 기능 상세

### 1. 최근 사용 그룹명 자동 입력

- 마지막으로 사용한 그룹명이 필드에 기본값으로 노출
- 매번 다시 입력할 필요 없이 바로 전시 올리기 가능

### 2. 자동완성 드롭다운

- 타이핑 시 내 이전 그룹명 + 서비스 전체 그룹명 목록 드롭다운 제공
- 기존 그룹명 선택 유도해 오탈자·띄어쓰기로 인한 그룹 분리 방지

### 3. 그룹명 정규화 (내부 로직)

- 저장 시 띄어쓰기·특수문자 제거 + 대소문자 통일 후 동일 그룹 여부 판단
- 표시는 최초 입력자의 원본 표기 방식 유지

| 케이스 | 처리 |
| --- | --- |
| "성동복지관 14기" (최초 입력) | 그룹 생성. 표시명: "성동복지관 14기" |
| "성동복지관14기" (이후 입력) | 동일 그룹으로 집계. 표시명은 최초 입력값 유지 |
| 자동완성에서 선택 | 동일 그룹으로 확실히 연결 |

## 미결 사항

- [x]  자동완성 드롭다운 기준 → **내 이전 그룹명 + 서비스 전체 그룹명 포함** ✅ (정책 모음 그룹명 처리 정책에서 확정)
- [x]  정규화 처리 범위 → **띄어쓰기 + 특수문자 제거 + 대소문자 통일** ✅
