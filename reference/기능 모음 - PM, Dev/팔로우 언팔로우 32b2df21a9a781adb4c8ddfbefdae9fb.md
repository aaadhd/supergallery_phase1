# 팔로우 / 언팔로우

관련 화면: 작가 프로필, 내 프로필, 작업물 상세 모달
화면 모음 - PM, PD, Dev v2: 내 프로필 (https://www.notion.so/3352df21a9a781278708ccc1dcf1cfcc?pvs=21), 팔로워 / 팔로잉 목록 (https://www.notion.so/3352df21a9a781729dc4f0a182e0ece4?pvs=21), 작가 프로필 (https://www.notion.so/3352df21a9a78182897bd669d7580490?pvs=21)

## 개요

작가를 팔로우하거나 언팔로우하는 기능. 작가 프로필, 작업물 상세 모달 등 여러 진입점에서 동작.

## 진입점

- 작가 프로필 페이지 내 팔로우 버튼
- 작업물 상세 모달 내 작가 영역
- 팔로워/팔로잉 목록 모달 내 카드

## 동작 정책

| 케이스 | 동작 |
| --- | --- |
| 비로그인 상태에서 클릭 | 로그인 유도 모달 |
| 팔로우 클릭 | 즉시 팔로우 처리 + 버튼 UI 변경 (팔로우 중 상태) |
| 언팔로우 클릭 | 확인 없이 즉시 언팔로우 |
| 자기 자신 프로필 | 팔로우 버튼 미노출 |
| 상대방이 이미 팔로우 중 | 버튼이 "팔로우 중" 상태로 표시 |

## 알림 연동

- 누군가 나를 팔로우하면 알림 발송 (알림 설정에서 ON/OFF 가능)

## 미결 사항

- [x]  언팔로우 시 확인 모달 → **확인 없이 즉시 언팔로우** ✅ (시니어에게 확인 단계는 불필요한 장벽. 실수로 누를 가능성 적음)
- [x]  맞팔 표시 여부 → **미표시** ✅ (복잡도만 올라감. Phase 2에서 검토)
