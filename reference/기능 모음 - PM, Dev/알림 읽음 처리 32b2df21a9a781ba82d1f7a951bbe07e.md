# 알림 읽음 처리

관련 화면: 알림
화면 모음 - PM, PD, Dev v2: 알림 툴팁 (https://www.notion.so/3352df21a9a78126bf76c92dcabdd025?pvs=21)

## 개요

알림의 읽음/안읽음 상태를 관리하는 기능.

## 동작 정의

| 기능 | 내용 |
| --- | --- |
| 단건 읽음 | 알림 클릭 시 해당 알림 읽음 체크 |
| 전체 읽음 | "모두 읽음으로 표시" 버튼 |
| 알림 삭제 | 개별 알림 삭제 ✅ 포함 (시니어 정리 수단 필요) |
| GNB 배지 | 안읽음 알림 있을 때 GNB 알림 아이콘에 뱃지 표시 |

## 알림 보관 정책

| 항목 | 내용 |
| --- | --- |
| 보관 기간 | 90일 |
| 최대 스톡 수 | 200개 (초과 시 오래된 알림 자동 삭제) |

## 미결 사항

- [x]  개별 알림 삭제 기능 Phase 1 포함 여부 → **포함** ✅ (알림이 쌓이면 시니어는 크게 부담스러워함. 정리 수단이 있어야 함)
- [x]  알림 보관 기간 및 최대 스톡 수 → **90일 / 200개** ✅ (알림 페이지에서 이미 확정)
