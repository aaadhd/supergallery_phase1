# 공지사항 / FAQ / 문의하기

## 개요

유저가 서비스 공지사항을 확인하고, FAQ로 자주 묻는 질문을 조회하며, 문의하기 폼으로 운영팀에 직접 문의를 접수할 수 있는 기능 묶음.

## 진입점

- GNB 또는 푸터 내 공지사항 / FAQ / 문의하기 링크
- 알림 목록 내 공지 알림 클릭 → 공지사항 상세 이동

## 동작 정책

### 공지사항 (/notices)

| 항목 | 내용 |
| --- | --- |
| 중요 공지 | 최대 3개 상단 고정 |
| 등록 / 수정 | 어드민에서 운영팀이 직접 관리 |

### FAQ (/faq)

| 항목 | 내용 | 구성 | 카테고리별 아코디언 형태 |
| --- | --- | --- | --- |
| 문항 | PM이 런칭 전 작성 | 수정 | 어드민 또는 코드 직접 수정 (Ralph 결정) |

### 문의하기 (/contact)

| 항목 | 내용 | 로그인 유저 | 이메일 / 닉네임 자동 입력 |
| --- | --- | --- | --- |
| 내용 | 최대 1,000자 | 파일 첨부 | 최대 5MB × 3개 |
| 제출 후 | 자동 응답 메일 발송 ("영업일 기준 3일 이내 답변") | 운영팀 | 문의 접수 시 운영팀 이메일 수신 |
