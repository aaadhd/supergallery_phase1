# 회원 관리 (어드민)

## 개요

운영팀이 어드민 대시보드에서 전체 회원 목록을 조회·검색하고, 계정 상태를 변경(활성/비활성)할 수 있는 기능. 신고 처리, 계정 정지·제재와 연동된다.

## 진입점

- 어드민 대시보드 → 회원 관리 메뉴

## 동작 정책

| 케이스 | 동작 |
| --- | --- |
| 회원 목록 조회 | 가입일 역순 정렬, 닉네임·이메일 검색 가능 |
| 계정 상태 변경 (비활성화) | 해당 유저 로그인 차단, 콘텐츠 피드 미노출 |
| 계정 재활성화 | 정상 상태 복구 |
| 어드민 접근 권한 | 어드민 전용 URL + 별도 인증 방식 (Ralph 결정) |
| 역할 기반 접근 제어 (RBAC) | 운영팀 Role만 회원 관리 메뉴 접근 가능 |

## 관련 EPIC

M: 어드민

## 개발 스펙

- AdminLog 테이블에 계정 상태 변경 이력 기록 (변경자, 변경 시각, 사유)
- 계정 비활성화 시 해당 유저의 기존 세션(JWT) 즉시 무효화 (Ralph 결정)
- 어드민 URL 및 인증 방식: Ralph 결정
