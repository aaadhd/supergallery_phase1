## 개요

카카오, 구글, 애플 소셜 계정으로 빠르게 가입 및 로그인하는 기능. GeoIP 국가 감지 결과에 따라 버튼 노출 우선순위가 달라진다.

## 진입점

- 회원가입 / 로그인 화면 상단 소셜 버튼 영역
- 비로그인 유도 모달 내 로그인 버튼 → 로그인 화면 이동 후 진입

## 동작 정책

| 케이스 | 동작 |
| --- | --- |
| 해외 방문자 | 구글·애플 버튼 최상단 노출 / 카카오 하단 |
| 기존 유저 | 자동 로그인 후 이전 페이지 복귀 (?redirect 파라미터 활용) |
| 동일 이메일 계정 중복 | **자동 연결 없이 기존 가입 방식으로 로그인 유도** (ADR 확정). 계정 연결은 Phase 2+ 계정 설정에서 명시적 연결로 제공 |

## 관련 EPIC

A: 인증/계정

## 개발 스펙

- TECH-A01 (OAuth 앱 등록 및 환경변수 세팅) 선행 필요
- TECH-A02 (GeoIP 국가 감지) 연동 필요
- Redirect URI, 동의 항목 설정은 각 플랫폼 개발자 콘솔에서 설정
