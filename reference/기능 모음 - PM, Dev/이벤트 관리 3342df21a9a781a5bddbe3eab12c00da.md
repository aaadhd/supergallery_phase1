# 이벤트 관리

관련 화면: 어드민 (관리자 기능), 이벤트 목록 / 이벤트 상세

> 어드민에서 이벤트를 등록·수정·종료하는 기능.
> 

> 유저 대상 이벤트 화면 스펙은 **화면 모음 > 이벤트 목록 / 이벤트 상세** 참조.
> 

---

## 기능 정의

| 기능 | 내용 |
| --- | --- |
| 이벤트 등록 | 제목 + 설명 + 배너 이미지 + 기간 + 상태 설정 |
| 이벤트 수정 | 진행 중 이벤트 내용 수정 가능 |
| 이벤트 종료 | 수동 종료 처리 or 기간 자동 종료 |
| 참여작 공개 여부 | 공개 (업로드 즉시 노출) / 비공개 (종료 시까지 미노출) 선택 |

## 이벤트 상태값

| 상태 | 설명 |
| --- | --- |
| `scheduled` | 예정 (시작일 이전) |
| `active` | 진행 중 |
| `ended` | 종료 |

## 이벤트 데이터 구조

| 필드 | 타입 | 설명 |
| --- | --- | --- |
| id | UUID | 자동 생성 |
| title | string | 이벤트 제목 |
| description | text | 이벤트 상세 설명 |
| banner_image_url | string | 배너 이미지 |
| start_at | datetime | 시작일 |
| end_at | datetime | 종료일 |
| status | enum | scheduled / active / ended |
| works_public | boolean | 참여작 공개 여부 |

## 참여작 공개 정책

| 설정 | 동작 |
| --- | --- |
| 공개 | 이벤트 참여 작품 업로드 즉시 이벤트 상세 페이지에 노출 |
| 비공개 | 이벤트 종료 후 일괄 공개 (결과 발표형 이벤트에 적합) |
