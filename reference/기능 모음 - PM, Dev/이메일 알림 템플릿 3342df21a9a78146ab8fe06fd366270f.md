# 이메일 알림 템플릿

관련 화면: 전체 (이메일 시스템)

> 시스템이 자동 발송하는 이메일 목록 및 내용 정의. 개발자 구현 기준 문서.
> 

> 발송 트리거 조건은 **기능 모음 > 이메일 알림 발송 조건** 참조.

## 📧 템플릿별 상세

### 1. 회원가입 환영

| 항목 | 내용 |
| --- | --- |
| 발송 시점 | 회원가입 완료 직후 |
| 제목 (한) | `Artier에 오신 것을 환영합니다 🎨` |
| 제목 (영) | `Welcome to Artier 🎨` |
| 본문 핵심 내용 | 환영 메시지 + 프로필 완성 유도 CTA + 첫 작품 업로드 유도 CTA |
| CTA 버튼 | "프로필 완성하기" → `/me/edit` |

### 2. 이메일 인증

| 항목 | 내용 |
| --- | --- |
| 링크 만료 | 30분 |
| 제목 (한) | `[Artier] 이메일 주소를 인증해주세요` |
| 제목 (영) | `[Artier] Please verify your email address` |

### 3. 비밀번호 재설정

| 항목 | 내용 |
| --- | --- |
| 링크 만료 | 1시간 |
| 제목 (한) | `[Artier] 비밀번호 재설정 링크입니다` |
| 보안 안내 | "이 링크를 요청하지 않으셨다면 무시하셔도 됩니다" 문구 필수 |

### 4~10. 기타 템플릿

> 상세 내용은 하단 이전 정책 모음 원본 문서와 동일. 발송 조건은 기능 모음 > 이메일 알림 발송 조건 참조.
>

## 🎨 이메일 공통 디자인 가이드

| 영역 | 내용 | 10종 공통 여부 |
| --- | --- | --- |
| 레이아웃 | 최대 너비 600px, 반응형 | ✅ 공통 |
| 로고 | Artier 로고 상단 배치 | ✅ 공통 |
| 폰트 | 시스템 폰트 (웹 폰트 미사용 권장) | ✅ 공통 |
| CTA 버튼 | 브랜드 컬러 적용, 텍스트·링크는 가변 | 🔄 가변 |
| 푸터 | 회사명 + 주소 + 수신 거부 링크 (마케팅 메일만) | ✅ 공통 |
| 다국어 | 유저 언어 설정에 따라 한/영 분기 발송 | ✅ 공통 |

> **디자인 프레임 1개.** 10종 모두 동일한 레이아웃 틀 사용. PD는 공통 프레임 1개만 Figma로 제작하면 됨.
