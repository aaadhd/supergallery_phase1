## 개요

작업물을 북마크하는 기능. 내 프로필 > 저장 탭에서 모아보기 가능.

## 진입점

- 작업물 상세 모달 우측 액션 영역
- 내 프로필 > 저장 탭

## 동작 정책

| 케이스 | 동작 |
| --- | --- |
| 비로그인 상태에서 클릭 | 로그인 유도 모달 |
| 로그인 상태에서 클릭 | 즉시 저장 적용 + 아이콘 filled 상태 전환 |
| 다시 클릭 | 저장 취소 |

> 저장 버튼 이 화면에서 스타일, 순서 등 UI 상세는 **화면 모음 > 작업물 상세 모달 페이지 참조.**
>
