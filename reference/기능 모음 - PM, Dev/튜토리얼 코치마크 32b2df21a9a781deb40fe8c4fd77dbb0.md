## 적용 화면 및 내용

### 1. 작업 업로드 — 코치마크 (최우선)

> 단계가 많고 시니어에게 가장 낯선 화면. 각 단계별로 순서대로 안내.
> 

| 단계 | 트리거 | 안내 문구 |
| --- | --- | --- |
| Step 1 | 업로드 화면 최초 진입 | "이미지를 여기에 끌어다 놓거나, 클릭해서 선택하세요" |
| Step 2 | 이미지 업로드 완료 후 | "이미지를 클릭하면 작품명과 작가를 설정할 수 있어요" |
| Step 3 | 우측 패널 배경색상 | "배경색과 여백은 전시 전체에 공통 적용돼요" |
| Step 4 | 다음 버튼 활성 시 | "'다음'을 누르면 전시 이름을 설정하고 올릴 수 있어요" |
- 각 코치마크는 이전 단계 완료 후 순서대로 노출
- 각 코치마크에 "확인" 버튼 또는 **5초** 후 자동 닫힘
- 전체 건너뛰기 버튼 제공 (우측 상단)

---

### 2. 온보딩 — 첫 업로드 유도 툴팁

> 온보딩 완료 후 둘러보기 페이지 첫 진입 시, GNB "작품 올리기" 버튼에 툴팁 1회 노출.
> 

| 트리거 | 위치 | 안내 문구 |
| --- | --- | --- |
| 온보딩 완료 후 둘러보기 페이지 첫 진입 | GNB "작품 올리기" 버튼 위 | "여기를 눌러 첫 작품을 올려보세요 🎨" |
- 툴팁 클릭 또는 3초 후 자동 닫힘
- 업로드 화면으로 이동하면 즉시 닫힘

---

### 3. 둘러보기 페이지 — 첫 진입 코치마크

> 피드 구조(탭, 카드 클릭, 그룹명 호버)를 처음 접하는 사용자 안내.
> 

| 트리거 | 위치 | 안내 문구 |
| --- | --- | --- |
| 둘러보기 페이지 최초 진입 | 탭 영역 (전체/개인전시/그룹 전시) | "탭을 눌러 원하는 전시 유형만 볼 수 있어요" |
| 작품 카드 첫 노출 | 첫 번째 카드 | "카드를 클릭하면 작품을 크게 볼 수 있어요" |
- 2단계 순차 노출
- "확인" 클릭 또는 화면 다른 곳 클릭 시 닫힘

---

### 4. 내 프로필 — 첫 진입 탭 안내

| 트리거 | 위치 | 안내 문구 |
| --- | --- | --- |
| 내 프로필 최초 진입 | 탭 네비게이션 | "올린 전시, 좋아요한 작품, 저장한 작품을 여기서 모아볼 수 있어요" |

### 5. 그룹 전시 생성 — 개념 안내 툴팁

| 트리거 | 위치 | 안내 문구 |
| --- | --- | --- |

- 화면 최초 진입 시 자동 노출
- "확인" 클릭 시 닫힘

## 공통 정책

| 항목 | 내용 |
| --- | --- |
| 노출 조건 | 해당 화면/기능 최초 진입 시 1회만 노출 |
| 노출 기록 저장 | 서버 계정에 저장 (기기 변경 후 재로그인해도 다시 표시 안 됨) |
| 재설정 | 계정 설정 페이지 하단 "튜토리얼 다시 보기" 버튼으로 초기화 가능 |
| 건너뛰기 | 작업 업로드 코치마크에는 전체 건너뛰기 제공. 나머지는 클릭/탭으로 닫기 |
| 애니메이션 | 부드러운 페이드인/아웃. 시니어 고려해 빠른 전환 없음 |
| 화면 이동 시 | 코치마크 노출 중 다른 화면으로 이동하면 즉시 닫힘 |
| 말풍선 위치 | 안내 대상 요소 기준 위/아래/옆 자동 계산. 화면 밖으로 벗어나지 않도록 처리 |
