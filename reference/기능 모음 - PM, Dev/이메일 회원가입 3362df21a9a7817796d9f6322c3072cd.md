# 이메일 회원가입

## 개요

이메일 + 비밀번호로 Artier 계정을 생성하는 기능. 가입 완료 전 이메일 인증이 필수이며, 인증 완료 시 자동 로그인된다.

## 진입점

- 회원가입 / 로그인 화면 내 "이메일로 가입" 버튼

## 동작 정책

| 케이스 | 동작 | 이메일 형식 오류 | 인라인 에러 메시지 즉시 노출 |
| --- | --- | --- | --- |
| 비밀번호 8자 미만 또는 영문+숫자 미포함 | 인라인 에러 메시지 즉시 노출 | 이미 가입된 이메일 | "이미 사용 중인 이메일입니다" 안내 |
| 인증 메일 발송 | 가입 후 즉시 발송, 유효 시간 30분 | 인증 링크 만료 | 재발송 버튼 노출 |
| 인증 완료 | 자동 로그인 후 온보딩 플로우 진입 | 만 14세 미만 | 가입 차단 + 안내 메시지 노출 |

## 관련 EPIC

A: 인증/계정

## 개발 스펙

- 비밀번호 최소 8자, 영문 + 숫자 조합 필수
- 이메일 인증 링크 유효 시간: 30분
- TECH-G01 (이메일 발송 서비스) 연동 필요
