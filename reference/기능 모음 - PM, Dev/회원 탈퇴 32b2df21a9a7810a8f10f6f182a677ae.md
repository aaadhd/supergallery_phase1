## 개요

회원이 서비스를 탈퇴하는 기능. 계정 설정에서 진입.

## 탈퇴 플로우

1. 계정 설정 > 탈퇴 진입
2. 탈퇴 사유 선택
3. 탈퇴 안내 모달 + **비밀번호 or 소셜 인증 재확인** (본인 확인용)
4. "탈퇴하기" 버튼 클릭 → 탈퇴 처리 완료
5. 로그아웃 처리 → 둘러보기 페이지 이동 (탈퇴 완료 이메일 자동 발송)

## 탈퇴 후 데이터 처리

| 항목 | 내용 |
| --- | --- |
| 작업물 | 익명 유지 ("작가 미상"으로 표시) |
| 닉네임 | 영구 사용 불가 (탈퇴 후에도 작품이 서비스에 남으므로 필명 보호) |
| 이메일 | 30일 후 재가입 가능 |
| 그룹전시 참여 작품 | 익명 유지 ("작가 미상의 작품"으로 표시). 전시에서 제거하지 않음 — 다른 참여자 전시 보호 |

## 탈퇴 유저 작품 노출 정책

| 항목 | 내용 |
| --- | --- |
| 작가명 표시 | **"작가 미상"** (EN: "Unknown Artist") |
| 프로필 이미지 | 기본 아바타로 대체 |
| 작가명 클릭 | 비활성 (링크 아님). 프로필 페이지 없음 |
| 둘러보기 피드 | 계속 노출 (검수 통과된 콘텐츠풀 유지) |
| 검색 | 작품 제목/태그로 검색됨. 작가명 검색에서는 제외 |
| 좋아요/저장 | 다른 유저가 계속 가능 |
| 그룹전시 | 작품 유지, 작가명만 "작가 미상"으로 대체 |

> "작가 미상"은 미술관에서 작자 불명 시 쓰는 표현. Artier의 디지털 갤러리 정체성과 부합하며, "탈퇴" 등 부정적 뉴앙스 없음.
>
