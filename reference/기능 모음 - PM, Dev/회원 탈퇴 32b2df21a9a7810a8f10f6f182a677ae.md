# 회원 탈퇴

관련 화면: 내 프로필, 계정 설정
화면 모음 - PM, PD, Dev v2: 계정 설정 (https://www.notion.so/3352df21a9a781d2ac51c0eeedc5d4e5?pvs=21)

## 개요

회원이 서비스를 탈퇴하는 기능. 계정 설정에서 진입.

## 탈퇴 플로우

1. 계정 설정 > 탈퇴 진입
2. 탈퇴 사유 선택
3. 탈퇴 안내 모달 + **비밀번호 or 소셜 인증 재확인** (본인 확인용)
4. "탈퇴하기" 버튼 클릭 → 탈퇴 처리 완료
5. 로그아웃 처리 → 발견 페이지 이동 (탈퇴 완료 이메일 자동 발송)

## 탈퇴 후 데이터 처리

| 항목 | 내용 |
| --- | --- |
| 작업물 | 익명 유지 ("삭제된 사용자"로 표시) |
| 닉네임 | 30일 후 재사용 가능 |
| 이메일 | 30일 후 재가입 가능 |
| 그룹전시 참여 작품 | 익명 유지 ("탈퇴한 작가의 작품"으로 표시). 전시에서 제거하지 않음 — 다른 참여자 전시 보호 |

## 미결 사항

- [x]  탈퇴 후 작업물 처리 방식 → **익명 유지** ✅ (영구 삭제면 그룹전시 무결성 및 다른 작가 작업물 소실 발생. Behance 동일 방식 — 탈퇴 후 작업자명은 "삭제된 사용자"로 표시)
- [x]  닉네임 재사용 가능 여부 → **30일 후 재사용 가능** ✅ (즉시 재사용은 닉네임 선점 리스크. 재가입 허용 시 닉네임 해제 처리)
- [x]  동일 이메일 재가입 제한 → **30일 후 재가입 가능** ✅ (충동 탈퇴 방지. 상시 차단은 이용약관으로 별도 처리)
