# 이미지 업로드 및 블록 편집

관련 화면: 작업 업로드

> 작업 업로드 화면에서 이미지 블록을 추가·편집하는 기능. 노트폴리오와 동일한 규격 적용.
> 

---

## 개요

이미지 또는 이미지 그리드 블록을 업로드하고, 각 블록을 편집하는 기능 모음.

## 진입점

- 작업 업로드 화면 > 이미지 추가 / 이미지 그리드 추가 클릭

> 이미지 블록 툴바 스펙 (업로드 전/후 툴바 구성, 콘텐츠 재정렬 방식) 상세는 **화면 모음 > 작업 업로드 페이지 참조.**
> 

## 업로드 정책

| 항목 | 내용 |
| --- | --- |
| 허용 파일 형식 | JPG, JPEG, PNG, WEBP, GIF — 업로드 시 WEBP로 자동 변환 |
| 장당 최대 용량 | 10MB |
| 최대 업로드 장수 | 20장 |
| 권장 형식 | WEBP |

## 미결 사항

- [x]  이미지 최대 업로드 장수 → **20장** ✅ (한 전시에 지나치게 많으면 시니어 피로도 증가. 추후 데이터 보고 조정)
- [x]  GIF 형식 → **포함. 업로드 시 WEBP로 자동 변환** ✅ (용량·성능 이슈 및 아트 플랫폼 정체성과 맞지 않음. Phase 2에서 재검토)
