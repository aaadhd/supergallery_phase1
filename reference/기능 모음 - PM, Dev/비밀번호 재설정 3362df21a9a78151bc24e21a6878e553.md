# 비밀번호 재설정

## 개요

이메일 가입 유저가 비밀번호를 잊었을 때 이메일로 재설정 링크를 받아 새 비밀번호를 설정하는 기능. 소셜 로그인 전용 계정에는 해당 없음.

## 진입점

- 로그인 화면 내 "비밀번호를 잊으셨나요?" 링크

## 동작 정책

| 케이스 | 동작 |
| --- | --- |
| 미가입 이메일 입력 | 동일한 안내 메시지 노출 (이메일 존재 여부 미노출, 보안상) |
| 재설정 링크 만료 (1시간 초과) | "링크가 만료되었습니다" 안내 + 재발송 유도 |
| 소셜 전용 계정 | 해당 소셜 서비스에서 비밀번호 변경 안내 |

## 관련 EPIC

A: 인증/계정

## 개발 스펙

- 재설정 링크 유효 시간: 1시간
- 링크 1회 사용 후 무효화
- 새 비밀번호 조건: 8자 이상, 영문 + 숫자 조합
- TECH-G01 (이메일 발송 서비스) 연동 필요

## 미결 사항

- [x]  미가입 이메일 처리 → **동일 메시지 노출 (계정 존재 여부 미노출)** ✅
- [x]  링크 유효 시간 → **1시간** ✅
