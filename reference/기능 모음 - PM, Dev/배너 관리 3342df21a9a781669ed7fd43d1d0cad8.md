# 배너 관리

관련 화면: 어드민 (관리자 기능), 둘러보기

> 둘러보기 피드 상단 배너 슬라이더 등록/수정/삭제 기능.
> 

> 어드민 화면 스펙은 **화면 모음 > 어드민 (관리자 기능)** 참조.
> 

---

## 기능 정의

| 기능 | 내용 |
| --- | --- |
| 배너 등록 | 이미지 업로드 + 제목 + 부제 + 링크 URL + 배지 (NEW/HOT/EVENT) |
| 배너 수정 | 기존 배너 내용 수정 |
| 배너 삭제 | 즉시 미노출 |
| 노출 순서 | 드래그앤드롭으로 순서 변경 |
| 노출 기간 | 시작일 ~ 종료일 설정 → 종료일 이후 자동 미노출 |
| 최대 배너 수 | **5개** |

## 배너 데이터 구조

| 필드 | 타입 | 설명 |
| --- | --- | --- |
| id | UUID | 자동 생성 |
| title | string | 배너 제목 |
| subtitle | string? | 배너 부제 (선택) |
| image_url | string | 배너 이미지 URL |
| link_url | string? | 클릭 시 이동 URL (선택) |
| badge | enum? | NEW / HOT / EVENT (선택) |
| order | int | 노출 순서 |
| start_at | datetime | 노출 시작일 |
| end_at | datetime? | 노출 종료일 (선택) |
| is_active | boolean | 수동 활성/비활성 |

## 노출 조건

- `is_active = true`
- `start_at <= 현재 시각`
- `end_at > 현재 시각` (end_at 없으면 무기한)
- 위 조건 충족하는 배너를 `order` 순으로 최대 5개 노출
