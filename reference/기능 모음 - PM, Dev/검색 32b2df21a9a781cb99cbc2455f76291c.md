## 개요

작품명·작가명·그룹명으로 작품과 작가를 검색하는 기능.

## 검색 대상 및 범위

| 검색 대상 | Phase 1 포함 |
| --- | --- |
| 작품명 | ✅ |
| 작가명 (닉네임) | ✅ |
| 태그 | ❌ Phase 1 미포함 |
| 전시명 | ✅ 포함 |
| 그룹명 | ✅ 포함 |

## 화면 동작

| 동작 | 내용 |
| --- | --- |
| GNB 검색바 입력 | 검색 결과 페이지로 이동 |
| 자동완성 | ✅ 지원 |
| 결과 없음 | 빈 상태 화면 + 안내 문구 |
| 검색어 유지 | 페이지 진입 후 URL에 검색어 유지 (`/search?q=`) |

## 미결 사항

- [x]  자동완성 지원 여부 → **지원** ✅ (시니어는 닉네임을 어떻게 쓰는지 모를 수 있음. 입력 중 후보 제시하면 타이핑 실수 줄여줌)
- [x]  전시명 검색 포함 여부 → **포함** ✅ (작품명만으로는 검색 범위가 너무 좁아. 전시명 포함으로 발견 가능성 높임)
- [ ]  결과 없음 화면 문구 (한/영) — 런칭 전 PM 작성
