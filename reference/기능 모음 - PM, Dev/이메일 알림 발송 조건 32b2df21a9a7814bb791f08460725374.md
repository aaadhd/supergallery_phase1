# 이메일 알림 발송 조건

관련 화면: 전체 (이메일 발송 트리거 기반)

## 개요

서비스 내 이벤트 발생 시 자동으로 이메일을 발송하는 조건 정의. 24번 이메일 알림 템플릿과 연결.

## 공통 발송 정책

- 수신 거부 링크 포함 필수 (마케팅 제외하고 모든 이메일)
- 동시 발송 제한: 야간 시간대 (22시~08시) 발송 회피 — 속보성 이메일은 예외
- 한/영 언어 모두 제작 필요
