## 개요

서비스 전반의 응답 속도와 로딩 성능 기준을 정의하고, 개발 전 Sprint에 걸쳐 지속적으로 준수해야 하는 비기능 요건.

## 성능 기준

| 항목 | 기준 |
| --- | --- |
| 이미지 업로드 처리 (10MB 기준) | 5초 이내 |
| API 평균 응답 시간 | 500ms 이내 |

## 적용 범위

- 전 Sprint에 걸쳐 적용 (특정 기능이 아닌 개발 전반 기준)
- 피드, 작업물 상세, 업로드, 검색 등 유저 핵심 동선 우선 적용

## 주요 구현 방향

- 이미지: WEBP 변환 + CDN 캐싱 (TECH-B01 파이프라인 연동)
- 피드: Infinite Scroll + 커서 기반 페이지네이션
- API: 주요 쿼리 인덱스 최적화 (TECH-O01 DB 스키마 설계 시 반영)
- 코드 스플리팅 및 Lazy Loading 적용
