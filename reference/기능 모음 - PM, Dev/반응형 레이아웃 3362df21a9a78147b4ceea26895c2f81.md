# 반응형 레이아웃

## 개요

PC와 모바일 환경 모두에서 불편함 없이 서비스를 사용할 수 있도록 반응형 레이아웃 기준을 정의. 전 화면에 적용되는 요건.

## 브레이크포인트 기준

| 구분 | 기준 | PC | 1,280px 이상 |
| --- | --- | --- | --- |
| 모바일 | ~767px | 태블릿 | Phase 1 별도 대응 없음 (PC/모바일 중 가까운 쪽 기준 적용) |

## 브라우저 지원 범위

- Chrome / Safari / Edge 최신 2년 이내 버전 지원
- IE 미지원

## GNB 반응형 기준

| 환경 | GNB 구성 |
| --- | --- |
| 모바일 | 햄버거 메뉴 없음 — 메뉴 항목 2개(둘러보기/이벤트)만 있어 텍스트 유지 가능 |

## 적용 범위

- 전 Sprint에 걸쳐 모든 화면에 적용
- 디자인 가이드: AX실 박금지 PD 산출물 기준 준수

## 관련 EPIC

N: 법적페이지/공통UI, O: 시스템/비기능
