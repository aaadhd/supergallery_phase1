## 공유 동작 정의

| 항목 | 내용 |
| --- | --- |
| 공유 트리거 | 그룹전시 상세 페이지 내 공유 버튼 클릭 |
| 공유 방식 | OS 기본 공유 시트 (Share Sheet) 활용 |
| 공유 대상 | 링크 (이미지 파일 다운로드 버튼 미제공) |
| OG 이미지 | 전시명 + 커버 이미지 + 개설자 이름으로 자동 생성 |
| 재공유 | 링크를 받은 사람도 동일한 방식으로 재공유 가능 |
| 다운로드 | 버튼 미제공 — 링크와 분리된 이미지 유통 방지 |
