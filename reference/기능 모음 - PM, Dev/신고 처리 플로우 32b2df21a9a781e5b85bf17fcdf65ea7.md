## ⚙️ 신고 처리 정책

| 항목 | 내용 |
| --- | --- |
| 중복 신고 | 동일 유저가 동일 대상 중복 신고 불가 (시스템 차단) |
| 자기 자신 신고 | 불가 |
| 신고 후 처리 | 즉시 삭제 아님 — 운영팀 검토 후 결정 |
| 처리 기준 | 커뮤니티 가이드라인 위반 여부 |
| 허위 신고 반복 | 1주일 이내 3회 이상 확인 시 경고, 지속 시 7일 차단 |
| 신고 후 피드 숨김 | 신고한 유저에게만 즉시 숨김 처리 |
| 신고 접수 확인 | **신고 완료 시 토스트 노출**: "신고가 접수되었습니다" (EN: "Your report has been submitted") |
| 신고 처리 결과 안내 | 신고자 별도 안내 안 함 |
| 처리 기한 | 접수 후 7일 이내. 불법 콘텐츠 24시간 이내 우선 처리 |

## 🔄 운영팀 처리 플로우

```
유저 신고 접수
     ↓
어드민 신고 목록에 노출
     ↓
운영팀 검토 (작품/프로필 확인)
     ↓
  ┌──────────┬──────────┐
위반 없음     경고       삭제·계정 제재
  ↓           ↓            ↓
신고 기각   작가 알림   콘텐츠 삭제
```

## ⚙️ 개발 스펙

**Report 테이블**

```
Report (
  id, reporter_id,
  target_type [work/user],
  target_id,
  reason, detail,
  status [pending/reviewed/rejected/actioned],
  created_at, reviewed_at, reviewed_by
)
```

| 항목 | 내용 |
| --- | --- |
| 중복 방지 | (reporter_id + target_type + target_id) Unique 제약 |
| 어드민 노출 | 신고 목록 — 최신순 정렬, 상태별 필터 |
