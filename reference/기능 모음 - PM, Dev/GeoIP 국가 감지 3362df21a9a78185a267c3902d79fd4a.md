# GeoIP 국가 감지

## 개요

방문자의 접속 국가를 IP 기반으로 자동 감지하여 소셜 로그인 버튼 노출 우선순위와 기본 언어를 분기하는 기능.

## 진입점

- 회원가입 / 로그인 화면 최초 진입 시
- GNB 언어 설정 미선택 상태의 첫 방문 시

## 동작 정책

| 감지 결과 | 소셜 버튼 우선순위 | 기본 언어 |
| --- | --- | --- |
| 해외 (KR 외) | 구글·애플 최상단 / 카카오 하단 | 영어 (EN) |

## 기술 방식

- GeoIP 라이브러리 또는 CDN Edge (Cloudflare 등) 활용
- 클라이언트 먼저 체크 후 서버로 폴백 가능 (Ralph 결정)
- VPN 우회 시 완벽한 감지는 보장하지 않음 (베스트 에포트)

## 관련 EPIC

A: 인증/계정, N: 법적페이지/공통UI

## 개발 스펙

- TECH-A01 (소셜 OAuth 앱 등록) 완료 후 연동
- 유저가 GNB에서 언어를 수동 변경 시 GeoIP 감지 결과보다 수동 설정 우선 적용

## 미결 사항
