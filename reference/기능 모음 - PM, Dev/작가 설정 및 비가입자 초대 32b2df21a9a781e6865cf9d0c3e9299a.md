# 작가 설정 및 비가입자 초대

관련 화면: 작업 업로드
화면 모음 - PM, PD, Dev v2: 작업 업로드 (https://www.notion.so/3352df21a9a781a2ba7fcbd716998c2f?pvs=21)

> 업로드 시 각 이미지의 작업자를 지정하고, 비가입자에게 게시 즉시 초대 안내를 발송하는 기능.
> 

---

## 개요

강사가 수강생 작품을 대신 올리거나, 여러 명의 작품을 한 번에 올릴 때 각 이미지에 실제 작업자를 지정할 수 있음. Artier에 가입되지 않은 작업자에게는 게시 즉시 초대 메시지가 발송된다.

## 진입점

- 작업 업로드 > 장별 메타데이터 패널 > 작가 필드

## 작가 설정 플로우

```jsx
작가 필드 클릭
  ↓
닉네임 또는 아이디 입력
  ├── 가입자 검색됨 → 드롭다운에서 선택 → 완료
  └── 결과 없음 → "직접 입력" 전환
            ↓
        이름(닉네임) 입력 + 전화번호 입력
            ↓
        전시 올리기 완료 시 초대 메시지 자동 발송
```

## Artier 가입 작가 검색

| 항목 | 내용 |
| --- | --- |
| 검색 대상 | Artier 가입 유저 |
| 검색 기준 | 닉네임 또는 아이디 |
| 결과 표시 | 프로필 이미지 + 닉네임 드롭다운 |
| 선택 시 | 해당 유저로 작업자 지정 |

## 미가입 작가 직접 입력

| 항목 | 내용 |
| --- | --- |
| 입력 필드 | 이름(닉네임) + 전화번호 |
| 전화번호 용도 | 초대 메시지 발송용 — 발송 완료 후 즉시 삭제 |
| 발송 시점 | 전시 올리기 완료 즉시 |
| 발송 채널 | (한국) 카카오 알림톡 우선, SMS 보조 / (한국 외) SMS |

## 초대 메시지 내용

| 항목 | 내용 |
| --- | --- |
| 발신 | Artier 공식 번호 또는 카카오 채널 |
| 메시지 형태 | "[작업자 이름]님의 그림이 Artier에 전시되었습니다! 지금 바로 확인해보세요 👉 [전시 링크]" |
| 링크 | 해당 전시 페이지 URL |
| 수신자 | 업로드 시 입력된 전화번호 |

## 동작 정책

| 항목 | 내용 |
| --- | --- |
| 미입력 시 | 업로드한 본인으로 자동 표시 |
| 발송 동의 | 업로드 화면에 "전시 올리기 완료 시 작업자에게 초대 안내를 보냅니다" 안내 문구 표시 (별도 체크박스 없음) |
| 중복 발송 방지 | 동일 전화번호 동일 전시에 대해 재발송 방지 |
| 발송 실패 시 | 게시는 정상 처리, 발송 실패 알림 운영팀 수신 |
| 전화번호 보관 | 발송 완료 후 즉시 삭제 (개인정보처리방침 반영) |
| 미가입 수신자 링크 진입 | 서비스 소개 + 가입 유도 CTA 포함 랜딩페이지 진입 |

## 미결 사항

- [x]  초대 발송 채널 → **(한국) 카카오 알림톡 우선, SMS 보조 / (한국 외) SMS** ✅
- [ ]  초대 메시지 문구 한/영 최종 확정 (런칭 전 PM 작성)
- [x]  발송 동의 UI → **안내 문구만 표시** ✅ (별도 체크박스는 시니어에게 피로감)
- [x]  전화번호 보관 여부 → **발송 완료 후 즉시 삭제** ✅ (개인정보 최소 보유 원칙)
- [x]  발송 실패 처리 → **게시 정상 처리, 운영팀 수신** ✅
- [x]  미가입 수신자 링크 진입 → **서비스 소개 + 가입 유도 랜딩페이지** ✅
