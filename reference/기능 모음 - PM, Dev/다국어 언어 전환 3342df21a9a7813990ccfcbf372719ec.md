## 언어 전환 동작

| 항목 | 내용 |
| --- | --- |
| 진입점 | GNB 우측 지구본 언어 전환 버튼 |
| 저장 방식 | 로그인 유저: 서버 저장 (User.language) / 비로그인: localStorage |
| 전환 범위 | 현재 페이지 전체 UI 즉시 적용 |
| 기본값 | 브라우저 언어 설정 감지 우선. 감지 안 될 시 한국어 기본 |
| 이메일 발송 | 유저 언어 설정(User.language) 에 따라 한/영 분기 발송 |

## 개발 스펙

| 항목 | 내용 |
| --- | --- |
| User.language | `ko` / `en` |
| URL 구조 | 언어 prefix 없음 (`/` 기준. i18n 라이브러리 구현 방식은 개발자 결정) |
| i18n 라이브러리 | 개발자 결정 (next-intl / i18next 등) |
| 번역 내용물 관리 | JSON 파일 기반. PM이 한/영 문구 작성 후 제공 |
