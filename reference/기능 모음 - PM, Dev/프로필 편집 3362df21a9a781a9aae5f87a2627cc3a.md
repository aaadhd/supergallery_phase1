# 프로필 편집

## 개요

가입 유저가 닉네임, 프로필 이미지, 한 줄 소개, 국가, 관심 분야, 외부 링크(SNS 등)를 언제든 수정할 수 있는 기능.

## 진입점

- 내 프로필 페이지 → 프로필 편집 버튼
- 온보딩 플로우 Step 1 (최초 가입 시)

## 동작 정책

| 항목 | 규칙 |
| --- | --- |
| 프로필 이미지 | JPG/PNG/WEBP, 최대 5MB, 정방형 크롭 권장 |
| 국가 | 드롭다운 선택, 선택 입력 |
| 외부 링크 | Instagram / YouTube / X / 기타 URL, 최대 3개 |
| 저장 | 저장 즉시 반영, 별도 승인 없음 |

## 관련 EPIC

A: 인증/계정, F: 작가프로필

## 개발 스펙

- 프로필 이미지 업로드: TECH-B01 (이미지 업로드 파이프라인) 연동
- 닉네임 중복 체크: 실시간 검증 (디바운스 적용)
