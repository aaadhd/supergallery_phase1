## 개요

가입 유저가 닉네임, 프로필 이미지, 한 줄 소개, 국가, 관심 화풍, 외부 링크(SNS 등)를 언제든 수정할 수 있는 기능.

## 진입점

- 내 프로필 페이지 → 프로필 편집 버튼
- 온보딩 플로우 Step 1 (최초 가입 시)

## 동작 정책

| 항목 | 규칙 |
| --- | --- |
| 프로필 이미지 | JPG/PNG/WEBP, 최대 5MB, 정방형 크롭 권장 |
| 국가 | 드롭다운 선택, 선택 입력 |
| 외부 링크 | Instagram / YouTube / X / 기타 URL, 최대 3개 |
| 관심 화풍 | 드롭다운 복수 선택, 선택 입력 (8개 항목). 프로필에 태그로 표시 |
| 저장 | 저장 즉시 반영, 별도 승인 없음 |

## 관심 화풍 목록 (Phase 1 확정)

> 디지털 툴로 표현 가능한 화풍/기법 기준. 프로크리에이트 브러시 라이브러리 기준 분류.
> 

> 프로필에 태그로 표시. Phase 2에서 필터/개인화 신호로 활용 예정.
> 

> 향후 미술/패션/공예/제품디자인 등 상위 "분야" 레이어가 추가되면, 이 목록은 "미술" 하위로 배치.
>
