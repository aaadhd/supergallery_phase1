# 업로드 화면 — 콘텐츠 재정렬 모드

관련 화면: US-B01, US-B03
상위 항목: 작업 업로드 (%EC%9E%91%EC%97%85%20%EC%97%85%EB%A1%9C%EB%93%9C%203352df21a9a781a2ba7fcbd716998c2f.md)
선택: 작업 업로드
선택 1: 서브
화면ID: SCR-UPLOAD-05

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 업로드된 블록 순서를 드래그앤드롭으로 변경한다 |
| 진입 방법 | 업로드 화면 > 재정렬 모드 진입 (PD 결정) |
| 관련 유저스토리 | US-B01 |

## 🧩 구성

- 전체 블록 목록 (드래그 가능 상태)
- 각 블록에 핸들 아이콘 표시
- 재정렬 완료 버튼

## ⚡ 액션

- 블록 드래그앤드롭으로 순서 변경
- 재정렬 완료 → 편집 모드 복귀

## 🔁 상태

- 재정렬 모드 활성
