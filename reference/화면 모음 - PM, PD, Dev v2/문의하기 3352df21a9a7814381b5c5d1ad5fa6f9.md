# 문의하기

URL: /contact
선택: 푸터
선택 1: 화면
진입 방법: 푸터 링크
화면ID: SCR-CONTACT-01

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 유저가 운영팀에 직접 문의를 남긴다 |
| 진입 방법 | 푸터 > 문의하기 / FAQ 하단 링크 |
| 관련 유저스토리 | US-N02 |

## 🧩 구성

- 문의 유형 선택 (드롭다운: 계정 / 업로드 / 기타)
- 제목 입력
- 내용 입력 (최대 500자)
- 이메일 입력 (답변 수신용 — 로그인 시 자동 완성)
- 제출 버튼

## ⚡ 액션

- 문의 유형 선택
- 내용 입력 후 제출
- 제출 완료 → 완료 안내 메시지

## 🔁 상태

- 기본 / 에러(필수 항목 누락) / 제출 완료

## 🖼 미니 와이어프레임

```
────────────────────
  문의하기
────────────────────
문의 유형 [드롭다운 ▼]
제목      [입력]
내용      [입력, 최대 500자]
이메일    [자동 완성]
────────────────────
        [제출]

(제출 완료)
  ✅ 문의가 접수됐어요
  영업일 기준 2~3일 내
  답변 드릴게요.
```
