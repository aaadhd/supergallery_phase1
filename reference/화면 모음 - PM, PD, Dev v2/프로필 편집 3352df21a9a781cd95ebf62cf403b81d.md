## 📄 화면 기획

### 편집 가능 항목

| 필드 | 필수/선택 | 제한 | 비고 |
| --- | --- | --- | --- |
| 프로필 이미지 | 선택 | JPG, PNG / **최대 5MB** | 기본 아바타 유지 가능 |
| 닉네임 | 필수 | **최대 20자** / 중복 불가 | **한글·영문·숫자·_·- 허용, 공백·특수문자 불가** |
| 한줄 소개 | 선택 | **최대 20자** |  |
| 국가 | 선택 | 드롭다운 |  |
| 관심 화풍 | 선택 | **최대 5개** | 파인아트/일러스트/수채화/유화/드로잉/풍경화/인물화/정물화/추상화/사진/디지털아트/조각/공예/패션/제품 디자인 |
| 소개 | 선택 | 자유 입력 | **3줄 이상 접힘 + 더보기** |
| 외부 링크 | 선택 | URL 형식 검증 | 인스타그램·블로그 등 |

### 저장 동작

| 케이스 | 동작 |
| --- | --- |
| 저장 성공 | 토스트 메시지 "저장됐어요" + 내 프로필(`/me`)로 이동 |
| 닉네임 중복 | "이미 사용 중인 닉네임이에요" 에러 메시지 |
| 닉네임 형식 오류 | 입력 규칙 안내 메시지 |
| 취소 클릭 | 변경 내용 버리고 내 프로필로 복귀 (**확인 모달 없음** — 시니어 단계 최소화) |
