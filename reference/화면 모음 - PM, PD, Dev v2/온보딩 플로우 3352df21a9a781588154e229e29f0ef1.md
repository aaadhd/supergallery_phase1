## 🎯 온보딩 목표

| 단계 | 목표 행동 | 이유 |
| --- | --- | --- |
| 1단계 | 프로필 완성 (이미지 + 한줄소개) | 플랫폼 정체성 형성 + 포인트 30 AP 적립 |
| 2단계 | 첫 작품 업로드 | 콘텐츠 생산자로 전환 + 포인트 100 AP 적립 |

> 💡 두 단계 모두 완료하면 최대 **180 AP** 적립 (가입 50 + 프로필 30 + 첫 업로드 100)
> 

> Phase 2에서 포인트 공개 시 "처음부터 쌓여 있었다"는 경험 제공

## 📄 화면별 스펙

### 환영 화면 (가입 완료 직후)

| 항목 | 내용 |
| --- | --- |
| 노출 시점 | 회원가입 완료 → 자동 이동 |
| 구성 | Artier 로고 + 환영 메시지 + CTA 버튼 |
| 메시지 (한) | "아티어에 오신 것을 환영해요 🎨 나만의 작품을 올리고, 발견되고, 연결되세요." |
| 메시지 (영) | "Welcome to Artier 🎨 Share your work. Get discovered. Connect." |
| CTA 버튼 | "시작하기" → Step 1으로 이동 |
| 건너뛰기 | "나중에 하기" → 발견 페이지 이동 (**허용**) |

---

### Step 1 — 프로필 설정

| 항목 | 내용 |
| --- | --- |
| 진행 표시 | 상단 진행 바 (1/2) |
| 입력 필드 | 프로필 이미지 업로드 + 한줄소개 입력 |
| 프로필 이미지 | 클릭 시 업로드 / 기본 이미지 제공 |
| 한줄소개 | 최대 **20자** |
| 완료 버튼 | "다음" → Step 2 이동 |
| 건너뛰기 | "나중에 하기" → Step 2 이동 (**허용**) |
| 완료 시 포인트 | 30 AP 백그라운드 적립 (Phase 1, UI 미노출) |

---

### Step 2 — 첫 작품 업로드 유도

| 항목 | 내용 |
| --- | --- |
| 진행 표시 | 상단 진행 바 (2/2) |
| 구성 | 일러스트 + 설명 문구 + CTA 버튼 |
| 설명 문구 (한) | "첫 작품을 올려보세요. 나의 이야기를 세상에 선보이는 첫 걸음이에요." |
| CTA 버튼 | "작품 올리기" → `/upload` 이동 |
| 건너뛰기 | "나중에 하기" → 발견 페이지 이동 (**허용**) |
