> 

### 회원가입/로그인 플로우

**이메일 가입 플로우**

1. 이메일 입력
2. 비밀번호 입력
3. 약관 동의
4. 이메일 인증 (선택)
5. 가입 완료 → **온보딩 프로세스 진입** (작가명/기본 프로필 설정)

**간편가입 플로우 (카카오/휴대폰/구글/애플)**

1. 간편가입 버튼 클릭
2. 소셜 인증 완료
3. 약관 동의 (소셜 첫 가입 시만 표시)
4. 가입 완료 → **온보딩 프로세스 진입** (작가명/기본 프로필 설정)

> ⚠️ 최초 로그인 후 온보딩 프로세스는 가입 방식(이메일/소셜) 무관하게 **반드시 진행** — 작가명(닉네임) 및 기본 프로필 설정 완료 후 서비스 이용 가능
> 

### 입력값 유효성

| 항목 | 규칙 | 에러 메시지 |
| --- | --- | --- |
| 이메일 | 이메일 형식 | "올바른 이메일 주소를 입력해주세요" |
| 비밀번호 | 최소 8자, 영문+숫자 조합 필수, 특수문자 선택 | *(작성 필요)* |
| 닉네임 | 2~20자, 한글·영문·숫자·언더스코어(_)·하이픈(-) 허용, 공백·특수문자 불가 | *(작성 필요)* |
| 중복 이메일 | 가입된 이메일 | "이미 사용 중인 이메일입니다" |
| 중복 닉네임 | 사용 중인 닉네임 | "이미 사용 중인 닉네임입니다" |

### 약관 동의 항목

| 항목 | 필수/선택 | 비고 |
| --- | --- | --- |
| 전체 동의 | — | 하위 항목 일괄 체크 |
| 이용약관 동의 | **필수** | 약관 전문 링크 `/terms` |
| 개인정보 수집·이용 동의 | **필수** | 방침 전문 링크 `/privacy` |
| 만 14세 이상입니다 | **필수** | 법적 의무 — 미체크 시 가입 불가 |
| 마케팅 정보 수신 동의 (이메일) | 선택 | 기본값 미체크 |
| 마케팅 정보 수신 동의 (푸시 알림) | 선택 | 기본값 미체크 |
- 필수 항목 미동의 시 가입 버튼 비활성화
- 동의 이력은 서버에 기록 (법적 증빙용)
- 소셜 로그인 첫 가입 시에도 동일한 약관 동의 화면 표시
