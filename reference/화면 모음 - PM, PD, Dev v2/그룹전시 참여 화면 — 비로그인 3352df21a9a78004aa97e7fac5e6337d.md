## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 그룹전시 참여 초대 링크를 받은 사용자가 전시를 확인하고 가입으로 유입된다 |
| 진입 방법 | 카카오 알림톡/SMS 초대 링크 클릭 (비로그인) |
| 관련 유저스토리 | US-C02 |

## 🧩 구성

- 전시 미리보기 (작품 이미지, 전시명)
- 가입 유도 CTA 배너 ("Artier에 가입하고 나의 작품을 확인해보세요")
- 가입하기 버튼
- 그냥 보기 링크 (비로그인 감상만)
- 링크 만료 안내 (7일 경과 시)

## ⚡ 액션

- 가입하기 → 회원가입 플로우 진입 → 완료 후 해당 전시 복귀
- 그냥 보기 → 비로그인으로 전시 감상

## 🔁 상태

- 유효 (비로그인) / 만료 (7일 경과)
