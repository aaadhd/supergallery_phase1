| 항목 | 내용 |
| --- | --- |
| 모달 vs 전용 페이지 | 모달 + URL 변경 (history.pushState) |
| 진입 경로 | 둘러보기 페이지 내 작품 카드 클릭 / 개인전시, 그룹전시 내 작품 카드 클릭 |
| 뒤로가기 동작 | 모달 닫기 + 둘러보기 페이지 스크롤 위치 유지 |
