# 회원가입 화면

관련 화면: US-A01, US-A02
상위 항목: 회원가입 / 로그인 (%ED%9A%8C%EC%9B%90%EA%B0%80%EC%9E%85%20%EB%A1%9C%EA%B7%B8%EC%9D%B8%203352df21a9a7812dae95ce3738e1782c.md)
선택: 회원가입/로그인
선택 1: 화면
화면ID: SCR-AUTH-01

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 화면 ID | SCR-AUTH-01 |
| 목적 | 신규 사용자가 이메일로 계정을 생성한다 |
| 진입 방법 | 로그인 화면 > 회원가입 이동 링크 |
| 관련 유저스토리 | US-A02 |

## 🧩 구성

- 상단 타이틀 ("회원가입")
- 이메일 입력 필드
- 비밀번호 입력 필드 (최소 8자, 영문+숫자)
- 닉네임 입력 필드 (2~20자)
- 약관 동의 영역 (전체 동의 + 개별 5개 필수 + 1개 선택)
- 가입하기 버튼
- 로그인 이동 링크

## ⚡ 액션

- 가입하기 (필수 약관 동의 + 모든 필드 유효 시 활성)
- 이메일 인증 링크 발송 (가입 완료 후)
- 로그인 화면 이동

## 🔁 상태

- 기본 / 유효성 에러 (중복 이메일, 닉네임 형식 오류 등) / 로딩
