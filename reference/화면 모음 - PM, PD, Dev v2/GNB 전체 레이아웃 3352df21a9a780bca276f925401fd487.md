## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 모든 페이지 상단에 고정 노출되는 글로벌 네비게이션 바 전체 구조 |
| 진입 방법 | 전 페이지 공통 노출 |
| 관련 유저스토리 | US-N06, US-N04 |

## 🧩 구성

**비로그인**

| 위치 | 요소 |
| --- | --- |
| 좌측 | 로고, 둘러보기, 이벤트 |
| 우측 | 작품 올리기, 언어 전환(KO/EN), 로그인 버튼 |

**로그인**

| 위치 | 요소 |
| --- | --- |
| 좌측 | 로고, 둘러보기, 이벤트 |
| 우측 | 작품 올리기, 알림 아이콘(배지), 아바타, 언어 전환(KO/EN) |

**모바일** — 텍스트 메뉴 유지 (햄버거 없음)

## ⚡ 액션

- 로고 클릭 → 둘러보기(/)
- 둘러보기 클릭 → /
- 이벤트 클릭 → /events
- 작품 올리기 클릭 → /upload (비로그인 시 로그인 유도 모달)
- 알림 아이콘 클릭 → 알림 툴팁
- 아바타 클릭 → 드롭다운
- 언어 전환 → KO/EN 즉시 전환

## 🔁 상태

- 비로그인 / 로그인 / 모바일
