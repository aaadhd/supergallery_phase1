## 📄 화면 기획

### 업로드 플로우

1. GNB "작품 올리기" 클릭 (로그인 필요)
2. 콘텐츠 유형 선택 (이미지 / 이미지 그리드)
3. 이미지 업로드 — 드래그앤드롭 or 파일 선택
4. 블록별 툴바로 패딩 변경 / 이미지 교체 / 순서 변경 / 삭제
5. 전시 공통 스타일 설정 (배경색상, 콘텐츠 간격)
6. 장별 설정 — 썸네일 클릭 시 패널에서 작품명·작가 설정
7. 화면 미리보기 확인 *(선택)*
8. **"다음" 버튼 클릭 → 세부 정보 설정 모달 자동 오픈**

> **이벤트 연결 업로드:** 이벤트 상세 페이지 "참여하기" 버튼 클릭 후 업로드 플로우 진입 시, 세부 정보 설정 모달에 해당 이벤트가 자동 연결된 상태로 표시됨. 별도 선택 없이 발행 시 자동 이벤트 참여 처리.
> 
1. 커버 이미지, 전시명, 그룹명, 원작 확인 체크
2. **전시 올리기** → 비가입 작업자에게 초대 발송 → 전시 페이지 이동

> 이미지 블록 툴바 스펙 (업로드 전/후, 그리드 블록, 재정렬 모드) 상세는 **기능 모음 > 이미지 업로드 및 블록 편집 참조.**
>

| 필드 | 필수/선택 | 비고 |
| --- | --- | --- |
| 커버 이미지 | 자동 적용 | 첫 번째 이미지 기본 적용. 변경 가능 |
| 수강생 작품 대신 올리기 | 선택 (토글) | 강사용 |
| 전시명 | 필수 | 최대 50자 |
| 그룹명 | 선택 (일반) / **필수 (강사 업로드 시)** | 자동완성 제공. 그룹명 처리 규칙 참조. 강사 업로드 시 필수 입력. |
| 본인이 직접 창작한 원작임을 확인합니다 | **필수 체크** | 미체크 시 발행하기 비활성 |

> **저작권:** Phase 1은 All Rights Reserved 단일 정책 적용. CCL 선택 UI는 Phase 2에서 제공 예정. Phase 1에서는 저작권 관련 안내 문구 UI 미노출.

> **Phase 1 제약:** 작품 1장당 작업자 1명만 태그 가능. 1장에 여러 명 연결하는 공동 소유는 Phase 2+.
> 

> **제거된 항목:** 카테고리, 태그 (Phase 2), 1장 다중 작업자 태그 (Phase 2+)

### 이미지 업로드 정책 (노트폴리오 기준 참조)

| 항목 | 내용 |
| --- | --- |
| 허용 파일 형식 | JPG, JPEG, PNG, WEBP *(GIF 포함 여부 결정)* |
| 장당 최대 용량 | 10MB |
| 최대 업로드 장수 | 20장 |
| 권장 형식 | WEBP (원활한 업로드 및 참여를 위해 권장) |
| 커버 이미지 | 첫 번째 이미지 자동 적용. 세부 정보 모달에서 수동 변경 가능 |
| 임시저장 | 지원 (초안 탭에서 확인 가능) |

> 💡 디지털 드로잉 중심 운영. 원화 업로드는 Phase 0 운영 결과 후 수요 확인 뒤 결정.
> 

> 🚨 **카메라 사진 업로드 차단** — EXIF 메타데이터 분석으로 카메라로 촬영한 파일 자동 차단. 기능 모음 > 업로드 파일 검증(사진 차단) 참조.
> 

> 그룹명 처리 규칙 (자동 입력, 자동완성, 정규화) 상세는 **기능 모음 > 그룹명 자동완성 및 정규화 참조.**
