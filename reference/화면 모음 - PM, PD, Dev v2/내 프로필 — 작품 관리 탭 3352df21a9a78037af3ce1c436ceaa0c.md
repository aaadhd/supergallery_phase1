## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 내가 작업자로 등록된 모든 작품을 확인한다 |
| 진입 방법 | 내 프로필 > 작품 관리 탭 |
| 관련 유저스토리 | US-F01, US-B03 |

## 🧩 구성

- 작품 카드 그리드
    - 내가 직접 올린 작품: `...` 메뉴 (수정 / 비공개로 설정 / 삭제)
    - 남이 올리고 내가 태그된 작품: 메뉴 없음 (조회만)
- **검수 상태 라벨** (본인에게만 노출)
    - `pending` → **"검수 중"** / "Under Review" (회색 배경, 카드 좌상단)
    - `approved` → 라벨 없음 (정상 카드)
    - `rejected` → **"게시 불가"** / "Not Approved" (빨간 배경). 클릭 시 거절 사유 안내 모달
- 빈 상태: "아직 작품이 없어요"

> 검수 상태 상세 정책은 **기능 모음 > 콘텐츠 검토 (피드 편입) > 유저 대면 상태 표시** 참조.
> 

## ⚡ 액션

- 작품 카드 클릭 → 작업물 상세 모달
- `...` > 수정 → 업로드 수정 화면
- `...` > 비공개로 설정 → 피드에서 즉시 숨김
- `...` > 삭제 → 확인 다이얼로그 후 영구 삭제

## 🔁 상태

- 기본 / 빈 상태
