# 내 프로필 — 작품 관리 탭

상위 항목: 내 프로필 (%EB%82%B4%20%ED%94%84%EB%A1%9C%ED%95%84%203352df21a9a781278708ccc1dcf1cfcc.md)
선택: 마이프로필
선택 1: 화면
화면ID: SCR-MYPROFILE-02

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 내가 작업자로 등록된 모든 작품을 확인한다 |
| 진입 방법 | 내 프로필 > 작품 관리 탭 |
| 관련 유저스토리 | US-F01, US-B03 |

## 🧩 구성

- 작품 카드 그리드
    - 내가 직접 올린 작품: `...` 메뉴 (수정 / 비공개로 설정 / 삭제)
    - 남이 올리고 내가 태그된 작품: 메뉴 없음 (조회만)
- 빈 상태: "아직 작품이 없어요"

## ⚡ 액션

- 작품 카드 클릭 → 작업물 상세 모달
- `...` > 수정 → 업로드 수정 화면
- `...` > 비공개로 설정 → 피드에서 즉시 숨김
- `...` > 삭제 → 확인 다이얼로그 후 영구 삭제

## 🔁 상태

- 기본 / 빈 상태
