> 운영팀이 Artier를 운영하기 위해 반드시 필요한 관리자 기능 스펙
> 

> 어드민 없이는 Artier's Pick 선정, 배너 관리, 신고 처리, 파트너 배지 부여 불가
> 

> 
> 

> **관련 기능 모음:**
> 

> - 배지 시스템 / Artier's Pick 배지 부여·해제
> 

> - 신고 처리 플로우
> 

> - 작품 수정 / 삭제
> 

> - 포인트 기능 (Phase 2)
> 

> ⚠️ **콘텐츠 노출 정책 (Phase 1)**: 업로드 직후 본인 프로필에는 즉시 노출. 둘러보기 피드는 운영팀 검토 통과 후 노출.
> 

> - **검토 SLA**: 24시간 이내
> 

> - **검토 목적**: 초기 서비스에서 다음을 방지 — 저품질 콘텐츠 노출 / 브랜드 이미지 훼손 / 스팸 콘텐츠 / 광고 콘텐츠 / 부적절 콘텐츠
> 

> - **피드 노출**: Artier’s Pick → 상단 고정 / 일반 통과작 → 랜덤 배정. 추천 알고리즘은 좋은 콘텐츠 풀에서 작동해야 의미가 있음.
>

## 🔍 콘텐츠 검토

> Phase 1 둘러보기 피드 노출의 게이트키퍼. 업로드된 모든 전시는 운영팀 검토 후 피드 편입됨.
> 

| 기능 | 내용 |
| --- | --- |
| 검토 목록 | 최신순 정렬. 상태 필터 (대기/통과/거절) |
| 검토 항목 | 전시명, 작품 이미지, 작가 정보, 스팸/부적절 여부 확인 |
| 통과 | 둘러보기 피드 풀 편입 → 랜덤 노출 |
| 거절 | 피드 미노출 + 작가에게 안내 알림 |
| 진단 및 거절 | 메모 입력 가능 (내부용) |
| SLA | **24시간 이내** 검토 완료 |

## 🏆 Artier's Pick 관리

| 기능 | 내용 |
| --- | --- |
| Pick 목록 | 현재 주 / 이전 주 Pick 목록 |
| Pick 선정 | 작품 ID or URL 입력 → 배지 부여 + 피드 상단 노출 |
| Pick 해제 | 배지 제거 + 피드 정상 노출 전환 |
| 주간 자동 초기화 | 매주 월요일 자정 이전 주 Pick 아카이브 *(배지는 영구 유지)* |
| Pick 이력 | 날짜별 선정 작품 목록 조회 |

## 🖼 배너 관리

| 기능 | 내용 |
| --- | --- |
| 배너 등록 | 이미지 업로드 + 제목 + 부제 + 링크 URL + 배지 (NEW/HOT/EVENT) |
| 배너 수정 | 기존 배너 내용 수정 |
| 배너 삭제 | 즉시 미노출 |
| 노출 순서 | 드래그앤드롭으로 순서 변경 |
| 노출 기간 | 시작일 ~ 종료일 설정 → 종료일 이후 자동 미노출 |
| 최대 배너 수 | 5개 |

## 🎪 이벤트 관리

| 기능 | 내용 |
| --- | --- |
| 이벤트 등록 | 제목 + 설명 + 배너 이미지 + 기간 + 상태 (예정/진행/종료) |
| 참여작 공개 여부 | 공개 (업로드 즉시 노출) / 비공개 (종료 시까지 미노출) 선택 |
| 이벤트 수정 | 진행 중 이벤트 수정 가능 |
| 이벤트 종료 | 수동 종료 처리 or 기간 자동 종료 |
