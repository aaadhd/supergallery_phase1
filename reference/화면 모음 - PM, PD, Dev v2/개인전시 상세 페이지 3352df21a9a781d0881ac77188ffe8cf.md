## 📄 화면 기획

### 페이지 구성

| 영역 | 내용 |
| --- | --- |
| **작가 프로필 카드** | 프로필 이미지 + 닉네임 + 한줄 소개 + 팔로우 버튼. 클릭 시 작가 프로필 페이지 이동 |
| **전시 이미지** | 업로드된 이미지 세로 스크롤 표시. 배경색상·콘텐츠 간격 전시 설정 적용 |
| **전시 정보** | 전시명, 작품명, 설명, CCL 표시 |
| **액션 영역** | 좋아요 / 저장 / 공유 (전시 초대장 형식) |
| **모든 작업 목록** | 이 작가의 최신순 전시 노출 + "프로필 자세히 보기 >" 링크 |

> 개인전시에만 작가 프로필 카드 + 모든 작업 목록 노출.
> 

> 그룹전시는 하단 참여 작가 목록 섹션으로 대체.
>

## 🗒 미결 사항

- [x]  전용 페이지 vs 모달 방식 → **모달 + URL 변경 (history.pushState)** ✅ (작업물 상세 모달과 동일 방식으로 통일. 페이지 전환 없이 감상할 수 있음)
- [x]  모든 작업 목록 노출 수 → **최신 6개** ✅ (너무 많으면 페이지가 길다. 6개면 작가의 기탁을 보여주기 충분)
- [x]  비로그인 상태 액션 → **로그인 유도 모달 표시** ✅ (좋아요/저장/팔로우 클릭 시. 로그인 완료 후 액션 자동 실행)
