## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 첫 방문자에게 쿠키 사용 동의를 받는다 |
| 진입 방법 | 쿠키 동의 이력 없는 첫 방문 시 자동 노출 |
| 관련 유저스토리 | US-N07 |

## 🧩 구성

- 하단 고정 배너
- 안내 문구 ("이 사이트는 쿠키를 사용합니다")
- 전체 동의 버튼
- 필수만 동의 버튼
- 개인정보처리방침 링크

## ⚡ 액션

- 전체 동의 → 필수·선택 쿠키 모두 허용, 배너 닫힘
- 필수만 동의 → 필수 쿠키만 허용, 배너 닫힘
- 개인정보처리방침 → /privacy 이동 (배너 유지)
- 쿠키 철회: 푸터 '개인정보 설정' 링크 클릭 시 배너 재노출

## 🔁 상태

- 첫 방문 (배너 노출) / 동의 완료 (배너 미노출)
