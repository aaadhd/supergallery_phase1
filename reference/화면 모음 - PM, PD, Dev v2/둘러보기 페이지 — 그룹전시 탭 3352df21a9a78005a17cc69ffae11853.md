## 🧩 구성

- 탭: 전체 / 개인전시 / 그룹전시(선택)
- 작업물 카드 그리드 (그룹전시만 노출)
- 카드에 그룹명 또는 "여러 작업자" 표시

## ⚡ 액션

- 탭 전환
- 작업물 카드 클릭 → 그룹전시 상세 페이지
- 그룹명 호버 → 멤버 팝오버
- 무한스크롤
