# 온보딩 단계 화면

관련 화면: US-L01
상위 항목: 온보딩 플로우 (%EC%98%A8%EB%B3%B4%EB%94%A9%20%ED%94%8C%EB%A1%9C%EC%9A%B0%203352df21a9a781588154e229e29f0ef1.md)
선택: 온보딩
선택 1: 화면
화면ID: SCR-ONBOARD-02

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 화면 ID | SCR-ONBOARD-02 |
| 목적 | Step 1 프로필 설정 + Step 2 첫 업로드 유도를 처리한다 |
| 진입 방법 | 온보딩 시작 화면 > 시작하기 |
| 관련 유저스토리 | US-L01 |

## 🧩 구성

**Step 1 — 프로필 설정 (진행 바 1/2)**

- 진행 바
- 프로필 이미지 업로드 영역 (클릭 시 업로드, 기본 아바타 제공)
- 한줄 소개 입력 필드 (최대 20자)
- 다음 버튼
- 건너뛰기 링크

**Step 2 — 첫 업로드 유도 (진행 바 2/2)**

- 진행 바
- 일러스트 영역
- 안내 문구
- 작품 올리기 버튼
- 건너뛰기 링크

## ⚡ 액션

- 프로필 이미지 업로드
- 한줄 소개 입력
- 다음 → Step 2 이동 (30 AP 백그라운드 적립)
- 작품 올리기 → /upload 이동
- 건너뛰기 → 둘러보기 이동

## 🔁 상태

- 기본 / 이미지 업로드 완료 / 에러 (닉네임 중복)
