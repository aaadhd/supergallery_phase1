# 로그인 유도 모달

관련 화면: US-A01, US-A02, US-E01, US-E02, US-G01
선택: 기타
선택 1: 서브
화면ID: SUB-AUTH-01

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 화면 ID | SUB-AUTH-01 |
| 목적 | 비로그인 상태에서 로그인이 필요한 액션 시도 시 로그인을 유도한다 |
| 진입 방법 | 좋아요 / 저장 / 팔로우 / 작품 올리기 클릭 (비로그인) |
| 관련 유저스토리 | US-N06 |

## 🧩 구성

- 배경 딤 처리
- 타이틀 (액션에 따라 동적 변경 — "좋아요를 남기려면 로그인해주세요" 등)
- 서브 문구
- 소셜 로그인 버튼 (국가 감지 기반)
- 로그인 버튼
- 회원가입 버튼
- 닫기(X) 버튼

## ⚡ 액션

- 소셜 로그인 (로그인 완료 후 원래 액션 자동 실행)
- 로그인 화면 이동
- 회원가입 화면 이동
- 모달 닫기

## 🔁 상태

- 기본 (트리거 액션에 따라 문구 동적 변경)
