## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 작가가 올린 공개 전시 목록을 탭에서 보여준다 |
| 진입 방법 | 작가 프로필 > 전시 탭 (기본 탭) |
| 관련 유저스토리 | US-F02 |

## 🧩 구성

- 전시 필터 탭 (전체 / 개인 / 그룹 / 수업)
- 전시 카드 그리드 (공개 전시만)
- 빈 상태: "아직 전시가 없어요"

## ⚡ 액션

- 필터 탭 전환
- 전시 카드 클릭 → 전시 상세 페이지 이동

## 🔁 상태

- 기본 / 빈 상태
