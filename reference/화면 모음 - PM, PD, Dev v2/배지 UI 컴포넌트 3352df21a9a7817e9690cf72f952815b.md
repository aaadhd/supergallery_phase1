# 배지 UI 컴포넌트

선택: 기타

진입 방법: 작업물 카드, 작업물 상세 모달
화면ID: CMP-BADGE-01

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 서비스 내 배지 유형별 디자인 및 사용 기준을 정의한다 |
| 진입 방법 | 내부 디자인 참고용 (유저 직접 진입 없음) |
| 관련 유저스토리 | US-M01 |

## 🧩 구성

**배지 종류**

| 배지 | 표시 위치 | 노출 조건 |
| --- | --- | --- |
| Artier's Pick | 작품 카드 / 상세 | 어드민 선정 |
| 파트너 작가 | 닉네임 옆 | 어드민 부여 |
| 강사 | 닉네임 옆 | 강사 토글 ON |
