## 🔔 알림 유형별 스펙

| 유형 | 내용 | 기본값 | 클릭 시 이동 |
| --- | --- | --- | --- |
| 내 작품 좋아요 | "OO님이 내 작품을 좋아합니다" | ON | 해당 작품 모달 |
| 새 팔로워 | "OO님이 팔로우하기 시작했습니다" | ON | 해당 작가 프로필 |
| Artier's Pick 선정 | "내 작품이 이번 주 Artier's Pick에 선정됐어요!" | ON | 해당 작품 모달 |
| 그룹전시 태그 | "OO님이 전시에 초대했습니다" | ON | 해당 전시 페이지 |
| 팔로잉 작가 신작 | "OO 작가가 새 작품을 올렸습니다" | **OFF** (선택 동의) | 해당 작품 모달 |
| 주간 테마 발표 | "이번 주 테마: [테마명]" | **OFF** (선택 동의) | 둘러보기 |
| 시스템 공지 | 서비스 업데이트, 이벤트 안내 | ON (필수) | 공지 페이지 or 이벤트 |
