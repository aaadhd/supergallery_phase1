# 로그인 화면

관련 화면: US-A01, US-A02, US-A03, US-A04
상위 항목: 회원가입 / 로그인 (%ED%9A%8C%EC%9B%90%EA%B0%80%EC%9E%85%20%EB%A1%9C%EA%B7%B8%EC%9D%B8%203352df21a9a7812dae95ce3738e1782c.md)
선택: 회원가입/로그인
선택 1: 화면
화면ID: SCR-AUTH-02

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 화면 ID | SCR-AUTH-02 |
| 목적 | 기존 사용자가 이메일 또는 소셜 계정으로 로그인한다 |
| 진입 방법 | 서비스 진입 / 로그인 유도 모달 > 로그인 버튼 |
| 관련 유저스토리 | US-A01, US-A02, US-A03 |

## 🧩 구성

- 상단 로고
- 이메일 입력 필드
- 비밀번호 입력 필드
- 로그인 버튼
- 비밀번호 찾기 링크
- 구분선
- 소셜 로그인 버튼 (국가 감지 기반 — 한국: 카카오/휴대폰, 해외: 구글/애플)
- 언어 전환 버튼
- 회원가입 이동 링크

## ⚡ 액션

- 이메일 로그인
- 소셜 로그인 (카카오 / 구글 / 애플)
- 비밀번호 찾기 이동
- 회원가입 화면 이동

## 🔁 상태

- 기본 / 에러 (인증 실패, 미가입 이메일) / 로딩

## 🖼 미니 와이어프레임

```
────────────────────
      [로고]
────────────────────
이메일 입력
비밀번호 입력
      [로그인]
  비밀번호를 잊으셨나요?
────────────────────
         ── 또는 ──
[카카오 로그인]      ← 한국
[휴대폰으로 시작하기] ← 한국
────────────────────
[구글 로그인]        ← 해외
[Apple로 로그인]     ← 해외
────────────────────
계정이 없으신가요? 회원가입
           KO / EN
```
