# 비밀번호 재설정 화면

상위 항목: 회원가입 / 로그인 (%ED%9A%8C%EC%9B%90%EA%B0%80%EC%9E%85%20%EB%A1%9C%EA%B7%B8%EC%9D%B8%203352df21a9a7812dae95ce3738e1782c.md)
선택: 회원가입/로그인
선택 1: 화면
화면ID: SCR-AUTH-04

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 화면 ID | SCR-AUTH-04 |
| 목적 | 비밀번호를 잊은 사용자가 이메일 인증으로 재설정한다 |
| 진입 방법 | 로그인 화면 > 비밀번호 찾기 링크 |
| 관련 유저스토리 | US-A04 |

## 🧩 구성

**Step 1 — 이메일 입력**

- 안내 문구
- 이메일 입력 필드
- 재설정 링크 발송 버튼
- 로그인 화면 돌아가기 링크

**Step 2 — 발송 완료 안내**

- 이메일 발송 완료 메시지
- 재발송 버튼 (30분 만료 안내)

**Step 3 — 새 비밀번호 입력** (이메일 링크 클릭 후)

- 새 비밀번호 입력 필드
- 비밀번호 확인 입력 필드
- 변경 완료 버튼

## ⚡ 액션

- 재설정 링크 발송
- 재발송 요청
- 새 비밀번호 저장 후 로그인 화면 이동

## 🔁 상태

- 기본 / 에러 (미가입 이메일, 링크 만료) / 로딩 / 완료
