# 신고 모달

관련 화면: US-I01
구분: 모달
기능 모음: 신고 처리 플로우 (https://www.notion.so/32b2df21a9a781e5b85bf17fcdf65ea7?pvs=21)
선택: 둘러보기-작품상세
선택 1: 서브
진입 방법: 작품 카드 또는 상세 내 ... 메뉴
화면ID: SUB-WORK-02

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 부적절한 작품을 신고하고 운영팀에 처리를 요청한다 |
| 진입 방법 | 작품 카드 또는 작업물 상세 `...` 메뉴 > 신고 |
| 관련 유저스토리 | US-K01 |

## ⚡ 액션

- 신고 사유 선택
- 추가 설명 입력 (선택)
- 신고 제출
- 모달 닫기

## 🔁 상태

- Step 1 (사유 선택) / Step 2 (제출 완료)

## 📄 화면 구성

### Step 1 — 사유 선택

| 영역 | 내용 |
| --- | --- |
| 타이틀 | "이 작품을 신고하는 이유를 선택해주세요" |
| 신고 사유 목록 | 저작권 침해 / 부적절한 콘텐츠 / 스팸 / 허위 작업물 / 기타 |
| 추가 설명 | 선택 입력 (최대 200자) |
| 제출 버튼 | 사유 선택 시 활성화 |
| 닫기(X) | 모달 닫기 |

### Step 2 — 제출 완료

| 영역 | 내용 |
| --- | --- |
| 완료 메시지 | "신고가 접수되었습니다. 운영팀이 7일 이내 검토 후 처리합니다. (불법 콘텐츠는 24시간 이내)" |
| 닫기 버튼 | 모달 닫고 원래 화면 복귀 |
