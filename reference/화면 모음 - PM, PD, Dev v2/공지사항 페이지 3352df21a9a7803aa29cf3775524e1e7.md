# 공지사항 페이지

상위 항목: 공지사항 (%EA%B3%B5%EC%A7%80%EC%82%AC%ED%95%AD%203352df21a9a781d986eadf8031224fa6.md)
선택: 푸터
선택 1: 화면
화면ID: SCR-NOTICE-01

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 서비스 공지사항 전체 목록을 카테고리별로 탐색한다 |
| 진입 방법 | 푸터 > 공지사항 |
| 관련 유저스토리 | US-N01 |

## 🧩 구성

- 페이지 타이틀 ("공지사항")
- 카테고리 필터 탭 (전체 / 서비스 업데이트 / 점검 안내 / 정책 변경 / 이벤트)
- 공지 카드 목록 (카테고리 배지 + 제목 + 날짜 + 중요 공지 상단 고정)
- 페이지네이션 (20건씩)
- 빈 상태: "등록된 공지사항이 없습니다"

## ⚡ 액션

- 카테고리 탭 전환
- 공지 카드 클릭 → 공지사항 상세 이동
- 페이지네이션

## 🔁 상태

- 기본 / 빈 상태
