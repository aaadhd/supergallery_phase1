## 🧩 구성

- 페이지 타이틀 ("공지사항")
- 카테고리 필터 탭 (전체 / 서비스 업데이트 / 점검 안내 / 정책 변경 / 이벤트)
- 공지 카드 목록 (카테고리 배지 + 제목 + 날짜 + 중요 공지 상단 고정)
- 페이지네이션 (20건씩)
- 빈 상태: "등록된 공지사항이 없습니다"

## ⚡ 액션
