## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 둘러보기 메인 배너 슬라이더에 노출될 배너를 등록·수정·삭제·순서 변경한다 |
| 진입 방법 | 어드민 > 배너 관리 |
| 관련 유저스토리 | US-M01 |

## 🧩 구성

- 배너 목록 (현재 등록된 배너, 최대 5개)
- 배너 등록 폼: 이미지 업로드, 제목, 부제, 링크 URL, 배지(NEW/HOT/EVENT), 노출 기간(시작일~종료일)
- 드래그앤드롭으로 노출 순서 변경
- 수정 / 삭제 버튼

## ⚡ 액션

- 배너 등록 (최대 5개)
- 수정
- 삭제 → 즉시 미노출
- 드래그앤드롭으로 순서 변경
- 종료일 이후 자동 미노출

## 🔁 상태

- 기본 / 등록 폼 열림
