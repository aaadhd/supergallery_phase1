## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 이미지 블록 업로드 후 썸네일 선택 상태를 보여준다 |
| 진입 방법 | 이미지 블록 업로드 완료 상태 |
| 관련 유저스토리 | US-B01 |

## 🧩 구성

- 업로드된 이미지 표시
- 선택 상태 표시 (테두리 강조)
- 블록 툴바: 패딩 변경(↔), 이미지 교체, 순서 변경(↕), 삭제(🗑)
- 우측 장별 패널 (작품명, 작업자 태그)

## ⚡ 액션

- 이미지 썸네일 클릭 → 장별 패널 열기
- 패딩 변경
- 이미지 교체
- 순서 변경
- 블록 삭제

## 🔁 상태

- 기본 / 선택됨 (패널 열림)
