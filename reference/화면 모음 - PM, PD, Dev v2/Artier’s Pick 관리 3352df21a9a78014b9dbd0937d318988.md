## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | Artier's Pick 작품을 선정하고 배지를 부여·해제한다 |
| 진입 방법 | 어드민 > Artier's Pick |
| 관련 유저스토리 | US-M02 |

## 🧩 구성

- 현재 주 Pick 목록 / 이전 주 아카이브
- Pick 선정: 작품 ID 또는 URL 입력
- Pick 해제 버튼
- 주간 자동 초기화 안내 (매주 월요일 자정)

## ⚡ 액션

- Pick 선정 → 배지 부여 + 피드 상단 노출
- Pick 해제 → 배지 제거
- 이전 주 이력 조회
