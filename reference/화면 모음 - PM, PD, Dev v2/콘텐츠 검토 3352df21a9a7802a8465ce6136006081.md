## 🧩 구성

- 검토 목록 (최신순, 상태 필터: 대기/통과/거절)
- 각 항목: 전시명, 이미지, 작가 정보, 업로드 시간
- 통과 / 거절 버튼
- 거절 시 메모 입력란
- SLA: 24시간 이내

## ⚡ 액션

- 상태 필터 전환
- 통과 → 피드 노출
- 거절 → 피드 미노출 + 작가 알림
- 메모 입력 (내부용)
