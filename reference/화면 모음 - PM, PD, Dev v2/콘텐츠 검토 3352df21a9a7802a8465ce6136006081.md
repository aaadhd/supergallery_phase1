# 콘텐츠 검토

상위 항목: 어드민 (관리자 기능) (%EC%96%B4%EB%93%9C%EB%AF%BC%20(%EA%B4%80%EB%A6%AC%EC%9E%90%20%EA%B8%B0%EB%8A%A5)%203352df21a9a7814c9f69c8f89e5b18f0.md)
선택: 어드민
선택 1: 화면
화면ID: ADM-02

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 업로드된 전시를 운영팀이 검토하고 피드 노출 여부를 결정한다 |
| 진입 방법 | 어드민 > 콘텐츠 검토 |
| 관련 유저스토리 | US-M01 |

## 🧩 구성

- 검토 목록 (최신순, 상태 필터: 대기/통과/거절)
- 각 항목: 전시명, 이미지, 작가 정보, 업로드 시간
- 통과 / 거절 버튼
- 거절 시 메모 입력란
- SLA: 24시간 이내

## ⚡ 액션

- 상태 필터 전환
- 통과 → 피드 노출
- 거절 → 피드 미노출 + 작가 알림
- 메모 입력 (내부용)
