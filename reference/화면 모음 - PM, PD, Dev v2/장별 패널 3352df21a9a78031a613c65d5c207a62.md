# 장별 패널

관련 화면: US-B01
상위 항목: 작업 업로드 (%EC%9E%91%EC%97%85%20%EC%97%85%EB%A1%9C%EB%93%9C%203352df21a9a781a2ba7fcbd716998c2f.md)
선택: 작업 업로드
선택 1: 화면
화면ID: SUB-UPLOAD-01

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 이미지 썸네일 클릭 시 작품명·작업자를 장별로 설정한다 |
| 진입 방법 | 업로드 화면 > 이미지 썸네일 클릭 |
| 관련 유저스토리 | US-B01, US-C01 |

## 🧩 구성

- 작품명 입력 필드 (선택, 미입력 시 자동 처리)
- 작업자 추가 영역
    - 가입자: 닉네임 검색 드롭다운
    - 비가입자: 이름 + 연락처 직접 입력 전환
- 비가입자 초대 발송 안내 문구
- 강사 업로드 시 작업자 태그 필수 표시

## ⚡ 액션

- 작품명 입력
- 가입자 검색 및 태그
- 비가입자 직접 입력
- 패널 닫기

## 🔁 상태

- 기본 / 검색 중 / 비가입자 입력 모드

## 🖼 미니 와이어프레임

