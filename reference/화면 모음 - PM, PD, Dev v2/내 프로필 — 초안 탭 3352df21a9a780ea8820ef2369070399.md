## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 임시저장(비공개)된 전시 초안을 확인하고 이어서 편집한다 |
| 진입 방법 | 내 프로필 > 초안 탭 |
| 관련 유저스토리 | US-B02 |

## 🧩 구성

- 초안 카드 그리드 (최근 수정 순)
- 각 카드에 `...` 메뉴 (이어서 편집 / 삭제)
- 빈 상태

## ⚡ 액션

- 카드 클릭 → 업로드 편집 화면으로 이동
- `...` > 이어서 편집
- `...` > 삭제 → 영구 삭제
