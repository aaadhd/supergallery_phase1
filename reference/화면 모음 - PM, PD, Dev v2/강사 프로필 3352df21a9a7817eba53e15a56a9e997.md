URL: /:username
비고: 일반 작가 프로필과 동일 URL, is_instructor=true 시 렌더링 분기

선택 1: 그룹

> 강사 프로필은 일반 작가 프로필 (`/:username`)과 동일한 URL을 사용.
> 

> `is_instructor = true`인 유저 접근 시 추가 탭 및 UI 요소가 노출됨.
> 

> 강사 인증 방식: **Phase 1 자율 체크** (프로필 내 강사 토글 ON)
>

## 📄 화면 구성 (일반 프로필 대비 차이점)

| 항목 | 일반 작가 프로필 | 강사 프로필 |
| --- | --- | --- |
| 탭 구성 | 전시 탭만 | 전시 탭 + **수강생 작품 탭** 추가 |
| 닉네임 옆 배지 | 없음 | **강사 배지** 표시 |
| 팔로우 버튼 | ✅ | ✅ |

### 수강생 작품 탭

- 해당 강사가 올린 그룹전시 중 본인이 작업자로 태그되지 않은 작품(수강생 작품) 모음
- 각 작품 카드에 수강생 이름 표시
- 그룹명 클릭 시 해당 그룹전시 상세 페이지로 이동
