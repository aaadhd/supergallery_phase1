## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 계정을 영구 삭제하고 서비스를 탈퇴한다 |
| 진입 방법 | 계정 설정 > 회원 탈퇴 버튼 |
| 관련 유저스토리 | US-A06 |

## 🧩 구성

**탈퇴 확인 모달**

- 탈퇴 안내 문구 (작품 익명 유지, 복구 불가 안내)
- 비밀번호 또는 소셜 인증 재확인
- 탈퇴하기 버튼
- 취소 버튼

## ⚡ 액션

- 인증 확인 후 탈퇴하기 → 계정 삭제 + 로그아웃 + 둘러보기(/) 이동
- 취소 → 모달 닫기

## 🔁 상태

- 기본 / 인증 에러 / 처리 중
