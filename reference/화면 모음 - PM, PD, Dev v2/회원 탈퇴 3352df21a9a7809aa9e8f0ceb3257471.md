## 🧩 구성

**탈퇴 확인 모달**

- 탈퇴 안내 문구 (작품 익명 유지, 복구 불가 안내)
- 비밀번호 또는 소셜 인증 재확인
- 탈퇴하기 버튼
- 취소 버튼

## ⚡ 액션

- 인증 확인 후 탈퇴하기 → 계정 삭제 + 로그아웃 + 둘러보기(/) 이동
- 취소 → 모달 닫기

## 🔁 상태

- 기본 / 인증 에러 / 처리 중
