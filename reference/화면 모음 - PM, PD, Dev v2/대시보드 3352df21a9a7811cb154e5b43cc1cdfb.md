## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 어드민 진입 시 서비스 핵심 지표와 즉시 처리 필요 항목을 한눈에 파악한다 |
| 진입 방법 | 어드민 진입 시 기본 랜딩 화면 |
| 관련 유저스토리 | US-M01 |

## 🧩 구성

- 지표 카드 6개
    - 오늘 신규 가입 / 오늘 업로드 / 이번 주 MAU / 누적 작품 수
    - 미검토 콘텐츠 수 (빨간 배지 강조)
    - 미처리 신고 수 (빨간 배지 강조)

## ⚡ 액션

- 미검토 콘텐츠 카드 클릭 → 콘텐츠 검토 화면
- 미처리 신고 카드 클릭 → 신고 관리 화면

## 🔁 상태

- 기본 / 로딩
