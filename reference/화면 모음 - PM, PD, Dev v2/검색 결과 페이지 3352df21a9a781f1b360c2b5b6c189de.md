## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 작품명·작가명·전시명 키워드로 검색하고 결과를 확인한다 |
| 진입 방법 | GNB 검색바 클릭 |
| 관련 유저스토리 | US-D01, US-D02 |

## ⚡ 액션

- 검색어 입력 후 엔터 → 결과 노출
- 탭 전환 (전체 / 작품 / 작가)
- 작품 카드 클릭 → 작업물 상세 모달
- 작가 카드 클릭 → 작가 프로필
- 팔로우 버튼 클릭
- 최근 검색어 클릭 / 삭제

## 🔁 상태

- 기본 (결과 있음) / 결과 없음 / 로딩

## 📋 검색 결과 페이지 구성

### 탭 구조

| 탭 | 내용 | Phase 1 |
| --- | --- | --- |
| 전체 | 작품 + 작가 혼합 | ✅ |
| 작품 | 작업물 카드 목록 | ✅ |
| 작가 | 작가 프로필 카드 목록 | ✅ |

### 작품 결과 카드

- 썸네일 / 작품명 / 작가명 / 좋아요 수
- 클릭 시 작업물 상세 모달 오픈

### 작가 결과 카드

- 프로필 이미지 / 닉네임 / 한줄 소개 / 팔로우 버튼
- 클릭 시 프로필 페이지 이동

### 정렬

> Phase 1은 정렬 드롭다운 없음. 관련도 단일 정렬 고정. Phase 2에서 데이터 누적 후 추가 옵션 검토.
> 

| 기준 | 적용 | 비고 |
| --- | --- | --- |
| 관련도순 | ✅ 고정 | Phase 1 유일 정렬 기준 |
| 최신순 | ❌ Phase 2 | — |
| 인기순 | ❌ Phase 2 | — |

**관련도 판단 기준 (개발자 참고)**

| 매칭 위치 | 가중치 | 비고 |
| --- | --- | --- |
| 작가명 완전 일치 | 높음 | 사람을 찾는 가장 명확한 의도 |
| 전시명 완전 일치 | 높음 | 제목 정확 매칭 |
| 작품명 완전 일치 | 중간 | 개별 작품 단위 검색 |
| 그룹명 완전 일치 | 중간 | 클래스/그룹 단위 탐색 |
| 부분 일치 (전체) | 낮음 | 완전 일치보다 후순위 |
| 동점 시 fallback | — | 최신순 |
