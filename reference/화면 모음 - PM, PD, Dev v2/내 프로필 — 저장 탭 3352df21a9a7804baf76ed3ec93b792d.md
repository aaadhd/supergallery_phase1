## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 내가 저장한 작품 목록을 확인한다 |
| 진입 방법 | 내 프로필 > 저장 탭 |
| 관련 유저스토리 | US-E02 |

## 🧩 구성

- 저장한 작품 카드 그리드 (최신 저장 순)
- 빈 상태

## ⚡ 액션

- 작품 카드 클릭 → 작업물 상세 모달
- 상세 모달에서 저장 토글 OFF → 목록에서 즉시 제거

## 🔁 상태

- 기본 / 빈 상태
