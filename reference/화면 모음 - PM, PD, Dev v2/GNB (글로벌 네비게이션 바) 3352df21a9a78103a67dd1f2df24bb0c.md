## 📄 GNB 구성 요소

### 비로그인 상태

| 영역 | 요소 | 비고 |
| --- | --- | --- |
| 좌측 | 로고 | `/` 이동 |
| 좌측 | 둘러보기 텍스트 메뉴 | `/` 이동 |
| 좌측 | 이벤트 텍스트 메뉴 | `/events` 이동 |
| 우측 | 작품 올리기 버튼 | 클릭 시 로그인 유도 모달 |
| 우측 | 언어 전환 버튼 | 한/영 전환 |
| 우측 | 로그인 버튼 | `/login` 이동 |

### 로그인 상태

| 영역 | 요소 | 비고 |
| --- | --- | --- |
| 좌측 | 로고 | `/` 이동 |
| 좌측 | 둘러보기 텍스트 메뉴 | `/` 이동 |
| 좌측 | 이벤트 텍스트 메뉴 | `/events` 이동 |
| 우측 | 작품 올리기 버튼 | `/upload` 이동 |
| 우측 | 알림 아이콘 | `/notifications` 이동. 미읽음 시 숫자 배지 (99+) |
| 우측 | 아바타 | 클릭 시 드롭다운 오픈. 화면 모음 > GNB 아바타 드롭다운 참조 |
| 우측 | 언어 전환 버튼 | 한/영 전환 |

### 모바일

- 텍스트 메뉴 유지 (햄버거 없음) — 메뉴 수가 적어 공간 충분
- 검색바는 GNB 내 아이콘으로 축소 or 별도 행 *(PD 결정 필요)*

## 🔗 관련 화면 문서

- GNB 아바타 드롭다운 → 화면 모음 > GNB 아바타 드롭다운
- 검색 기능 → 화면 모음 > 검색
- 알림 → 화면 모음 > 알림
- 언어 전환 → 기능 모음 > 다국어 / 언어 전환
