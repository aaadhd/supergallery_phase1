## 📄 화면 기획

### 전시 유형 개념

| 유형 | 정의 | 예시 |
| --- | --- | --- |
| **개인전시** | 1인 작가가 업로드한 작업물 | 내 갤러리 |
| **그룹전시** | 업로드 시 작업자를 2인 이상 태그한 전시 | 성남복지관 봄 수업전, 수채화 동호회 정기전 |

### Phase 1 핵심 원칙

> **컨테이너(전시)만 공동으로 사용합니다.**
> 

> 각 작품의 소유자는 제출한 본인 1명입니다.
> 

> 작품 레벨의 공동 소유자 태그는 **Phase 2+ 검토**.
> 

```jsx
그룹전시: "성남복지관 봄 수업전" (컨테이너)
├── 작품 A — 제출자: 김영자
├── 작품 B — 제출자: 이종환
├── 작품 C — 제출자: 박정호
└── 작품 D — 제출자: 조가영
```

### 그룹전시 생성 방식

> 별도 생성 플로우 없음. **업로드 시 작업자를 여러 명 태그하면 자동으로 그룹전시.**
> 
1. GNB "작품 올리기" 클릭
2. 이미지 업로드 후 세부 정보 모달 진입
3. 장별 패널에서 작업자 1명 태그 (일반: 선택 / 강사: 필수, 회원/비회원 가능)
    - 업로드한 작업자가 2인 이상이면 자동으로 그룹전시로 분류
4. 그룹명 입력 (일반: 선택 / 강사: 필수)
5. 발행 → 그룹전시 상세 페이지 이동

> 비회원 작업자 태그 시 → "네 그림이 Artier에 게시됐어요. 확인해 볼래요?" 알림 발송 (카카오 알림톡 우선, SMS 보조). 수락 여부 무관하게 전시는 즉시 공개.
> 

### 진입 경로

| 경로 | 설명 |
| --- | --- |
| 둘러보기 > 그룹전시 탭 | 전체 그룹전시 목록 조회 |
| 내 프로필 > 전시 탭 | 내가 참여한 그룹전시 포함 |
| 전시 초대장 공유 링크 | 비회원에게 발송된 알림으로 진입 |

## 📋 그룹전시 상세 페이지 구성

> 개인전시 상세와 거의 동일한 구조. **하단에 단체명 + 참여 작가 목록 추가.**
> 

| 영역 | 내용 |
| --- | --- |
| 헤더 | 전시명 + 커버 이미지 |
| 작품 갤러리 | 참여자 전체 작품 그리드 (제출순) |
| 액션 버튼 | 좋아요 / 공유 (전시 초대장 형식) |
| **하단 — 단체명 + 참여 작가 목록** | 그룹명 or "여러 작업자" + 참여 작가 N명 |

### 하단 참여 작가 목록

| 항목 | 내용 |
| --- | --- |
| 상단 헤더 | 그룹명 (없으면 "여러 작업자") + 참여 작가 N명 |
| 멤버 카드 — 가입자 | 프로필 이미지 + 닉네임 + 한줄 소개 + 팔로우 버튼 |
| 멤버 카드 — 비가입자 | 업로드 시 입력한 이름만 표시. 팔로우 버튼 없음 |
| 비로그인 팔로우 클릭 | 로그인 유도 모달 |
