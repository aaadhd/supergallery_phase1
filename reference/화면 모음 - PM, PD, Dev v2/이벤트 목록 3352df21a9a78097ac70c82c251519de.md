선택: 이벤트
선택 1: 화면
화면ID: SCR-EVENT-01

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 진행 중·예정·종료 이벤트 목록을 탐색한다 |
| 진입 방법 | GNB 이벤트 링크 / 배너 클릭 |
| 관련 유저스토리 | US-J01 |

## 🧩 구성

- 배너 슬라이더 (최대 5장, 5초 자동 전환)
- 이벤트 카드 목록 (진행 중 / COMING SOON / 종료)
- 빈 상태

## ⚡ 액션

- 이벤트 카드 클릭 → 이벤트 상세 이동
- 배너 클릭 → 해당 이벤트 상세 이동

## 🔁 상태

- 기본 / 빈 상태
