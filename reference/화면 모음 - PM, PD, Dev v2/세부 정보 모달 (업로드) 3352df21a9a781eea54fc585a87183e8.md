## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 업로드 마지막 단계에서 전시명·작업자·그룹명 등 세부 정보를 입력한다 |
| 진입 방법 | 이미지 업로드 완료 후 [다음] 버튼 클릭 |
| 관련 유저스토리 | US-B01 |

## ⚡ 액션

- 전시명 입력
- 작업자 태그 (장별)
- 그룹명 입력
- 임시저장 → 초안 탭 이동
- 전시 올리기 → 발행

## 🔁 상태

- 기본 / 에러(필수 누락) / 로딩
