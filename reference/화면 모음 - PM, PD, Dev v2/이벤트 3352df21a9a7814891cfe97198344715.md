## 📄 화면 기획

### 이벤트 목록 (`/events`)

| 항목 | 내용 |
| --- | --- |
| URL | `/events` |
| 진입 경로 | GNB 좌측 이벤트 링크 / 메인 배너 클릭 |
| 접근 권한 | 비로그인 포함 누구나 |

**구성**

- 배너 슬라이더 (최대 5장, 5초 자동 전환)
- 이벤트 카드 목록
    - 상태: 진행 중 / COMING SOON / 종료
    - 종료된 이벤트는 목록에서 자동 숨김 (어드민 설정)

### 이벤트 상세 (`/events/:id`)

| 항목 | 내용 |
| --- | --- |
| URL | `/events/:id` |
| 진입 경로 | 이벤트 목록 카드 클릭 |
| 접근 권한 | 비로그인 포함 누구나 열람 가능. 참여(업로드)는 로그인 필요 |

**구성**

- 이벤트 제목 + 설명 (Rich Text)
- 기간 표시
- 참여하기 CTA 버튼 (Sticky, 하단 고정)
    - 클릭 시 업로드 에디터로 이동 + 해당 이벤트 자동 연결
    - 종료된 이벤트는 버튼 비활성화 + 안내 문구
    - 비로그인 클릭 시 로그인 유도 모달
- 참여작 갤러리 (업로드된 작품 목록)
