선택: 둘러보기
선택 1: 서브
진입 방법: 둘러보기 첫 진입 시 자동 노출 (1회)
화면ID: SCR-FEED-04

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 피드 첫 진입 유저에게 주요 기능 위치를 1회 안내한다 |
| 진입 방법 | 둘러보기 최초 진입 시 자동 노출 (서버에 노출 여부 저장, 재방문 미노출) |
| 관련 유저스토리 | US-D04 |

## 🧩 구성

- 배경 딤 처리
- 말풍선 코치마크 3~4개
    - GNB 작품 올리기 버튼 → "여기서 작품을 올릴 수 있어요"
    - 작품 카드 → "클릭하면 전시를 감상할 수 있어요"
    - 알림 아이콘 → "좋아요·팔로우 알림이 여기에 와요"
- 5초 자동 닫힘 or 화면 탭 시 즉시 닫힘
- 닫기(X) 버튼

## ⚡ 액션

- 화면 탭 → 즉시 닫힘
- 닫기(X) → 닫힘
- 5초 후 자동 닫힘
