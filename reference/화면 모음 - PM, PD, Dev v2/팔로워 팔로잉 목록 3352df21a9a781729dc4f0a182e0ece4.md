| 항목 | 내용 |

## 🧩 구성

- 모달 컨테이너 (배경 딤 처리)
- 탭: 팔로워 / 팔로잉
- 유저 카드 목록 (프로필 이미지, 닉네임, 한줄 소개, 팔로우 버튼)
- 빈 상태

## ⚡ 액션

- 탭 전환 (팔로워 ↔ 팔로잉)
- 유저 카드 클릭 → 해당 작가 프로필 이동
- 팔로우 / 언팔로우
- 닫기

## 🔁 상태

- 기본 / 빈 상태 (팔로워 0명 또는 팔로잉 0명)
