## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 업로드 시작 시 콘텐츠 유형(이미지/이미지 그리드)을 선택한다 |
| 진입 방법 | GNB 작품 올리기 클릭 |
| 관련 유저스토리 | US-B01 |

## 🧩 구성

- 페이지 타이틀 ("새 전시 만들기")
- 콘텐츠 유형 선택 버튼 2개
    - 이미지 (단일/다중 이미지 세로 배치)
    - 이미지 그리드 (격자 배치)
- 각 유형 설명 문구

## ⚡ 액션

- 이미지 선택 → 이미지 블록 업로드 화면으로 전환
- 이미지 그리드 선택 → 그리드 블록 업로드 화면으로 전환
