## 📄 화면 기획

### 404 Not Found

| 항목 | 내용 |
| --- | --- |
| 트리거 | 존재하지 않는 URL 접근 |
| 구성 | 일러스트 + 안내 문구 + 홈으로 버튼 |
| CTA | 홈으로 (둘러보기 이동) |
| 푸터 | 포함 |

### 500 Server Error

| 항목 | 내용 |
| --- | --- |
| 트리거 | 서버 오류 발생 시 |
| 구성 | 일러스트 + 안내 문구 + 새로고침 버튼 + 홈으로 버튼 |
| 푸터 | 포함 |

### 점검 중 (`/maintenance`)

| 항목 | 내용 |
| --- | --- |
| 트리거 | 서비스 점검 시 전체 페이지 대체 |
| 구성 | 일러스트 + 점검 안내 문구 + 예상 완료 시간 *(선택)* |
| 어드민 우회 | 어드민 계정은 `/admin` 직접 접근 가능 |
| 푸터 | 미포함 |
