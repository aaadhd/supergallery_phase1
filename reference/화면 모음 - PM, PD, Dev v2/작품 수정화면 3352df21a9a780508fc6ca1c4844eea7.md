## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 기존 업로드된 작품을 수정 모드로 재진입하여 이미지·제목·설명 등을 편집한다 |
| 진입 방법 | 내 프로필 — 작품 관리 탭 > ··· > 수정 |
| 관련 유저스토리 | US-B01, US-F05 |

## 🧩 구성

- 기존 업로드 화면(SCR-UPLOAD-01~06)과 동일한 레이아웃
- 상단 타이틀: "전시 수정하기"
- 기존 이미지·블록 구성이 로드된 상태로 진입
- 세부 정보 모달(SUB-UPLOAD-02) 내 기존 제목·설명·태그·공개 여부 값 자동 입력
- 하단 CTA: "저장하기" (게시 완료와 동일 동작)
- 우측 상단 "취소" 버튼 → 수정 내용 미저장 경고 모달 후 이전 화면 복귀

## ⚡ 액션

- 이미지 교체/추가/삭제 가능
- 블록 순서 변경 가능 (재정렬 모드 진입)
- 세부 정보 모달에서 제목·설명·태그·작업자 수정
- "저장하기" 클릭 → 수정 내용 즉시 반영 + 내 프로필 복귀
- "취소" 클릭 → 미저장 경고 모달 → 확인 시 수정 내용 폐기

## 🔁 상태

- 기본 (기존 콘텐츠 로드됨)
- 수정 중 (변경사항 있음)
- 저장 완료

## ⚠️ 제약 / 예외

- 그룹전시 제출작은 수정 불가 (수정 메뉴 비활성 또는 미노출)
- 비공개 처리된 작품도 수정 진입 가능
- 수정 중 페이지 이탈 시 미저장 경고
