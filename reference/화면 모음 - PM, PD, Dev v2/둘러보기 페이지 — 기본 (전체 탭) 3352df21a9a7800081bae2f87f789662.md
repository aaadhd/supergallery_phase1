상위 항목: 둘러보기 (%EB%91%98%EB%9F%AC%EB%B3%B4%EA%B8%B0%203352df21a9a78124b0ebc155fe84354c.md)

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 서비스 메인 피드 — 개인전시+그룹전시 전체를 랜덤 노출한다 |
| 진입 방법 | GNB 로고 / 둘러보기 링크 / 서비스 첫 진입 |
| 관련 유저스토리 | US-D01, US-L01 |

## 🧩 구성

- 배너 슬라이더 (최대 5장, 5초 자동전환)
- 탭: 전체(기본) / 개인전시 / 그룹전시
- 작업물 카드 그리드 (PC 4열 / 모바일 2열, 무한스크롤)
- 스켈레톤 UI (로딩 중)

## ⚡ 액션

- 탭 전환
- 배너 클릭 → 해당 전시/이벤트
- 작업물 카드 클릭 → 작업물 상세 모달
- 작가 이름 클릭 → 작가 프로필
- 좋아요 / 저장 토글 (비로그인 시 로그인 유도)
- 무한스크롤 → 다음 페이지 로드

## 🔁 상태

- 기본 / 로딩(스켈레톤) / 에러(재시도 버튼)
