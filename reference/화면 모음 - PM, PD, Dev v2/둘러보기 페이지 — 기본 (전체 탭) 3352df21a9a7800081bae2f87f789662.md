## 🧩 구성

- 배너 슬라이더 (최대 5장, 5초 자동전환)
- 탭: 전체(기본) / 개인전시 / 그룹전시
- 작업물 카드 그리드 (PC 4열 / 모바일 2열, 무한스크롤)
- 스켈레톤 UI (로딩 중)

## ⚡ 액션

- 탭 전환
- 배너 클릭 → 해당 전시/이벤트
- 작업물 카드 클릭 → 작업물 상세 모달
- 작가 이름 클릭 → 작가 프로필
- 좋아요 / 저장 토글 (비로그인 시 로그인 유도)
- 무한스크롤 → 다음 페이지 로드

## 🔁 상태

- 기본 / 로딩(스켈레톤) / 에러(재시도 버튼)
