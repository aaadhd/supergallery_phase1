상위 항목: 작업 업로드 (%EC%9E%91%EC%97%85%20%EC%97%85%EB%A1%9C%EB%93%9C%203352df21a9a781a2ba7fcbd716998c2f.md)

## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 발행 전 전시 화면을 실제 노출 형태로 미리 확인한다 |
| 진입 방법 | 업로드 화면 > 미리보기 버튼 |
| 관련 유저스토리 | US-B01 |

## 🧩 구성

- 전시 화면 실제 렌더링 (배경색·간격 적용)
- 상단 "미리보기" 레이블
- 뒤로가기 버튼 (편집 화면 복귀)
- 발행하기 버튼

## ⚡ 액션

- 뒤로가기 → 편집 화면 복귀
- 발행하기 → 세부 정보 모달 오픈
