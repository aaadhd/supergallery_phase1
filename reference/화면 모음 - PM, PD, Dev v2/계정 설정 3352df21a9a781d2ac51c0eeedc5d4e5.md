## 🚪 회원 탈퇴 플로우

1. 설정 페이지 하단 "회원 탈퇴" 버튼 클릭
2. 탈퇴 사유 선택
3. 탈퇴 안내 모달 노출
4. 비밀번호 또는 소셜 인증 재확인
5. "탈퇴하기" 버튼 클릭 → 탈퇴 완료
6. 로그아웃 처리 후 둘러보기 페이지 이동

### 탈퇴 안내 모달 문구 (안)

> "탈퇴하시면 계정 정보와 좋아요·팔로우 기록이 삭제됩니다. 올리신 작품은 작가명 없이 서비스에 남아 있으며 복구할 수 없습니다. 작품을 삭제하고 싶으시면 탈퇴 전에 내 프로필 > 작품 관리에서 직접 삭제해주세요.
> 

> 정말 탈퇴하시겠습니까?"
> 

> 탈퇴 후 데이터 처리 상세 (작업물 익명 유지, 법령 보존 기간 등) → **기능 모음 > 회원 탈퇴 참조**
>
