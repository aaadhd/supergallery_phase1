### 배너 슬라이더

| 항목 | 내용 |
| --- | --- |
| 슬라이드 수 | 최대 5장 |
| 자동 전환 | On |
| 전환 간격 | 5초 |
| 주요 활용 | 주간 테마 기획전 · 런칭 이벤트 등 안내 |
| 클릭 시 | 해당 전시/작업물/이벤트 페이지 이동 |
| 관리 주체 | 운영팀 수동 등록 |

### 탭 구조

| 탭 | Phase 1 노출 | 내용 |
| --- | --- | --- |
| 전체 | ✅ | 개인전시 + 그룹전시 전체 (기본 선택) |
| 개인전시 | ✅ | 개인 작가 작업물 |
| 그룹전시 | ✅ | 그룹 공동 작업물 |
| 카테고리 탭 (회화/패션/공예/제품디자인) | ❌ Phase 1 미노출 | 작품 수 충분해지면 순차 오픈 |

### 작업물 카드 구성 요소

- 썸네일 이미지
- **개인 전시:** 작가 프로필 이미지 + 닉네임 (클릭 시 프로필 페이지로 이동)
- **그룹전시:** 그룹명 (없으면 "여러 작업자") (호버 시 멤버 팝오버)
- 전시명
- 좋아요 수 · 조회수
- Artier's Pick 배지 (선정작에만 노출)
- Weekly Best 배지 (선정작에만 노출)

### 정렬 · 피드 정책

| 항목 | 내용 |
| --- | --- |
| 기본 정렬 | 랜덤 (이전에 본 전시 제외) |
| 무한스크롤 vs 페이지네이션 | 무한스크롤 |
| 업로드 후 피드 반영 | 운영팀 검토(24시간 이내) 통과 후 피드 노출. 본인 프로필에는 즉시 노출. |
| 콘텐츠 풀 노출 | Artier’s Pick 선정작 → 피드 상단 고정 / 일반 통과작 → 랜덤 배정. 이번 주 테마전·신규전시·작가추천 포함 |
| 업로드 후 유저 안내 | "업로드가 완료되었어요. 콘텐츠 검토 후 둘러보기 피드에 노출됩니다." |
| 내 프로필에서는 | 업로드 즉시 노출 |
