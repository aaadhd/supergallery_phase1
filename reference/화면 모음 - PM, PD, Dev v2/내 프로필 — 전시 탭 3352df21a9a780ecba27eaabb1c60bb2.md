## 🧩 구성

- 전시 카드 그리드 (업로드한 전시 목록)
- 각 카드에 `...` 메뉴 (수정 / 비공개로 설정 / 삭제)
- 빈 상태: "아직 전시가 없어요" + 첫 업로드 CTA

## ⚡ 액션

- 전시 카드 클릭 → 전시 상세 이동
- `...` > 수정 → 업로드 수정 화면
- `...` > 비공개 → 피드에서 숨김 처리
- `...` > 삭제 → 확인 다이얼로그 후 영구 삭제
