## 📋 화면 개요

| 항목 | 내용 |
| --- | --- |
| 목적 | 신고 접수 건을 검토하고 처리한다 |
| 진입 방법 | 어드민 > 신고 관리 |
| 관련 유저스토리 | US-M01 |

## 🧩 구성

- 신고 목록 (상태 필터: 대기/검토완료/처리됨)
- 신고 상세: 신고 내용, 대상 작품/유저 미리보기
- 처리 액션: 신고 기각 / 콘텐츠 삭제 / 작가 경고 / 계정 정지
- 처리 메모 (내부용)
- SLA: 일반 7일 / 불법 콘텐츠 24시간

## ⚡ 액션

- 필터 전환
- 처리 액션 선택
- 메모 입력 후 처리 완료
