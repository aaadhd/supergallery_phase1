## 1️⃣ 회원가입 / 로그인 플로우

### 신규 회원가입

> 휴대폰 번호 가입은 Phase 2 앱 출시 시 도입 (앱 전용). 웹은 이메일이 없으면 트랜잭셔널 알림 발송 불가하므로 소셜/이메일만 제공.
> 

```jsx
서비스 진입
  ▶ 회원가입 화면
    ◆ 국가 감지
      ├─ [한국] 카카오 / 이메일 선택
      └─ [해외] 구글 / 애플 / 이메일 선택
    ▶ 닉네임 입력 (2~20자)
    ▶ 약관 동의
    ▶ 가입 완료
      ✅ 온보딩 플로우 진입 (플로우 11)
```

### 이메일 가입 상세

```
이메일 입력
  ▶ 비밀번호 입력 (최소 8자, 영문+숫자)
    ▶ 이메일 인증 발송
      ◆ 인증 링크 클릭 (30분 이내)
        ├─ [성공] 닉네임 입력으로 이동
        └─ [만료] 재발송 안내
```

### 기존 유저 로그인

```jsx
로그인 화면
  ◆ 로그인 방식 선택
    ├─ 소셜 로그인 → ✅ 자동 로그인 후 둘러보기
    └─ 이메일/비밀번호 입력
        ◆ 인증 결과
          ├─ [성공] ✅ 둘러보기
          └─ [실패] 오류 메시지 → 비밀번호 찾기 안내
```

### 비밀번호 찾기

```
비밀번호 찾기 탭
  ▶ 이메일 입력
    ▶ 재설정 링크 발송 (1시간 이내)
      ◆ 링크 클릭
        ├─ [유효] 새 비밀번호 입력 → ✅ 로그인 화면
        └─ [만료] 재발송 안내
```

## 2️⃣ 작품 업로드 → 전시 생성 플로우

> **Phase 1 작업자 태그 정책**: 1장당 1명만 태그 가능. 1장에 여러 명 연결하는 공동 소유는 Phase 2+.
> 

> **강사 업로드 시**: 그룹명 필수 입력. 작업자(수강생) 태그는 장별 선택 (회원/비회원 모두 가능).
> 

```jsx
🔒 로그인 상태
  ▶ + 버튼 (GNB)
    ▶ 갤러리에서 이미지 선택 (최대 20장, 장당 10MB)
      ◆ 파일 형식 확인
        ├─ [허용: JPG/PNG/WEBP/GIF] 업로드 진행
        └─ [미허용] 오류 안내
      ▶ 장별 패널 설정 (썸네일 클릭 시)
          ▶ 작품명 입력 (선택)
          ▶ 작업자 태그 (1장당 1명, 선택)
            ◆ 작업자 유형
              ├─ [가입자] 닉네임 검색 → 태그
              └─ [비가입자] 이름 + 연락처 입력
                  ▶ "네 그림이 Artier에 게시됐어요" 알림 발송
                     (카카오 알림톡 우선 / SMS 보조)
          ▶ 강사 업로드 시: 그룹명 필수 입력 / 작업자 태그는 장별 선택
      ▶ 전시명 입력 (필수, 최대 50자)
      ▶ 그룹명 입력 (선택 / 강사 업로드 시 필수)
        ◆ 작업자 2인 이상 태그된 경우 → 그룹전시로 자동 분류
      ▶ 작품명 미입력 시 자동 처리
        ◆
          ├─ [1명, 1장] 전시명 자동 적용
          └─ [1명 여러 장 / 그룹전시] 무제
      ▶ 저작권: All Rights Reserved 자동 적용 (Phase 1 고정값, 선택 UI 없음)
        ◆ 게시 or 임시저장
          ├─ [게시] ✅ 전시 상세 페이지 이동
                    본인 프로필에는 즉시 노출
                    둘러보기 피드는 운영팀 검토(24시간 이내) 후 편입
          └─ [임시저장] 초안 탭에 저장
```

## 3️⃣ 둘러보기 → 작품 상세 → 반응 플로우

### 비로그인

```jsx
둘러보기
  ▶ 작품 카드 탭
    ▶ 작업물 상세 모달 오픈 + URL 변경
      ◆ 액션 시도
        ├─ [공유] 공유 URL 복사 → 토스트 노출 (비로그인도 가능)
        └─ [좋아요 / 저장] 🔒 로그인 유도 모달 노출 후 둘러보기 복귀
```

### 로그인

```jsx
둘러보기
  ▶ 작품 카드 탭
    ▶ 작업물 상세 모달 오픈
      ◆ 반응 선택
        ├─ [좋아요] 하트 토글 ON ↔ OFF
        ├─ [저장] 저장 토글 ON ↔ OFF
        ├─ [공유] 공유 URL 복사 → 토스트 노출
        └─ [신고] 신고 모달 오픈
            ▶ 사유 선택 (라디오)
            ▶ 추가 설명 입력 (선택, 최대 200자)
            ▶ 신고 완료 → "신고가 접수되었습니다" 토스트 노출 + 둘러보기에서 즉시 숨김 + 모달 2초 후 자동 닫힘
      ▶ 작가 닉네임 탭 → 작가 프로필 페이지
```

## 4️⃣ 그룹전시 생성 플로우

> 개인전시와 그룹전시는 Exhibition 단일 엔티티로 통합. type 필드(personal/group)로 구분. URL 모두 /exhibitions/:id 사용.
> 

> **별도 그룹전시 생성 플로우 없음.** 업로드 시 작업자를 2인 이상 태그하면 자동으로 그룹전시.
> 

> **Phase 1 작업자 태그 정책**: 1장당 1명만 태그 가능. 그룹전시 = 공동 창작이 아닌 개별 작가들의 작품을 묶어 전시하는 형태.
> 

> **강사 업로드 시**: 그룹명 필수 입력, 작업자(수강생) 태그는 장별 선택.
> 

```jsx
🔒 로그인 상태
  ▶ + 버튼 (GNB) → 업로드 에디터
    ▶ 이미지 업로드
    ▶ 장별 패널에서 작업자 태그 (1장당 1명, 선택)
        ◆ 작업자 유형
          ├─ [가입자] 닉네임 검색 → 태그
          └─ [비가입자] 이름 + 연락처 입력
              ▶ "네 그림이 Artier에 게시됐어요" 알림 발송
                 (카카오 알림톡 우선 / SMS 보조)
                 수락 여부 무관하게 전시 즉시 공개
        ◆ 작업자 2인 이상 태그된 경우 → 그룹전시로 자동 분류
    ▶ 세부 정보 모달
      ▶ 그룹명 입력 (선택 / 강사 업로드 시 필수)
      ▶ 전시 올리기
        ✅ 그룹전시 상세 페이지 이동
```

## 5️⃣ 내 프로필 관리 플로우

### 프로필 편집

```jsx
🔒 로그인 상태
  ▶ GNB 아바타 탭 → 내 프로필
    ▶ 프로필 편집 버튼
      ▶ 프로필 이미지 변경 (선택)
      ▶ 닉네임 수정 (2~20자)
      ▶ 한 줄 소개 수정
      ▶ 관심 화풍 선택 (선택 입력, 복수 선택 가능)
        ✅ 저장 완료 → 내 프로필 복귀
```

### 작품 관리

```jsx
내 프로필 → 작품 관리 탭
  ◆ 작품 구분
    ├─ [내가 올린 작품] ... 메뉴 → 수정 / 비공개로 설정 / 삭제
    │    ▶ [수정] 업로드 수정 화면으로 이동
    │    ▶ [비공개로 설정] status = hidden → 피드에서 즉시 제거 (본인 프로필에는 유지)
    │    ▶ [삭제] 확인 다이얼로그 → 영구 삭제
    └─ [태그된 작품] 조회만 가능 (... 메뉴 없음)
```

### 저장 목록 조회

```jsx
🔒 로그인 상태
  ▶ GNB 아바타 탭 → 내 프로필
    ▶ 저장 탭 선택
      ▶ 저장한 작품 카드 목록 노출
        ▶ 카드 탭 → 작업물 상세 모달 (플로우 3 동일)
          ◆ 저장 토글 OFF
            └─ 목록에서 즉시 제거
```

### 계정 설정 / 탈퇴

```jsx
GNB 아바타 드롭다운 → 계정 설정
  ◆ 탈퇴하기 선택
    ▶ 탈퇴 사유 선택
    ▶ 탈퇴 안내 모달 (작품 익명 유지 안내)
    ▶ 본인 인증 재확인
    ▶ 탈퇴 완료
      ✅ 로그아웃 → 둘러보기 (비로그인 상태)
```

## 1️⃣1️⃣ 온보딩 플로우

> 회원가입 완료 직후 최초 1회만 진행. 건너뛰기 가능. 미완료 항목은 재접속 시 1회 재노출.
> 

```jsx
회원가입 완료
  ▶ 환영 화면 ("아티어에 오신 것을 환영해요 🎨")
    ◆ 건너뛰기 or 시작하기
      ├─ [건너뛰기] 둘러보기 이동 (미완료 항목 재접속 시 1회 재노출)
      └─ [시작하기]
          ▶ Step 1 — 프로필 설정 (진행 바 1/2)
            ▶ 프로필 이미지 업로드 (선택)
            ▶ 한 줄 소개 입력 (선택, 최대 20자)
              ◆ 건너뛰기 or 다음
                ├─ [건너뛰기] Step 2로 이동
                └─ [다음] 30 AP 백그라운드 적립 → Step 2로 이동
          ▶ Step 2 — 첫 작품 업로드 유도 (진행 바 2/2)
            ◆ 건너뛰기 or 작품 올리기
              ├─ [건너뛰기] ✅ 둘러보기 이동
              └─ [작품 올리기] /upload 이동 (플로우 2 진입)
```

## 1️⃣2️⃣ 비가입자 초대 링크 진입 플로우

> 강사 또는 업로더가 비가입자를 작업자로 태그 시, 카카오 알림톡/SMS로 초대 링크 발송.
> 

> 수락 개념 없음. 전시는 이미 공개 상태. 링크 유효기간 7일.
> 

```jsx
카카오 알림톡 / SMS 수신
  ▶ "네 그림이 Artier에 게시됐어요" 링크 클릭
    ◆ 링크 유효 여부
      ├─ [만료: 7일 경과] "링크가 만료되었습니다. 전시를 직접 검색해보세요." 안내
      └─ [유효]
          ◆ 로그인 여부
            ├─ [로그인 상태] 해당 그룹전시 상세 페이지 정상 로딩
            └─ [비로그인 상태]
                ▶ 전시 미리보기 + 가입 유도 CTA 노출
                  ("Artier에 가입하고 나의 작품을 확인해보세요")
                  ◆ 행동 선택
                    ├─ [가입하기] 회원가입 플로우 진입 (플로우 1)
                    │    ▶ 가입 완료 → 온보딩 (플로우 11)
                    │    ▶ 온보딩 완료 → 해당 그룹전시 상세 페이지 복귀
                    └─ [그냥 보기] 비로그인으로 전시 감상만 가능
```

## 1️⃣3️⃣ 이벤트 참여 플로우

> 이벤트 상세 페이지에서 작품을 업로드하면 해당 이벤트에 자동 연결.
> 

```jsx
진입 경로
  ◆ 진입 방법
    ├─ [GNB] 이벤트 메뉴 탭
    └─ [배너] 둘러보기 배너 클릭
  ▶ 이벤트 목록 (/events)
    ◆ 로그인 여부 (목록/상세 조회)
      ├─ [비로그인] 이벤트 목록 및 상세 조회 가능 (참여하기 버튼만 비활성)
      └─ [로그인] 전체 기능 사용 가능
    ▶ 이벤트 카드 탭
      ▶ 이벤트 상세 (/events/:id)
        ◆ 이벤트 상태
          ├─ [종료] "참여가 마감된 이벤트입니다" → 참여하기 버튼 비활성
          └─ [진행 중]
              ◆ 로그인 여부
                ├─ [비로그인] 참여하기 클릭 → 로그인 유도 모달
                └─ [로그인]
                    ▶ 참여하기 (Sticky CTA)
                      ▶ 업로드 에디터 진입 (플로우 2와 동일)
                        + 세부 정보 모달에 해당 이벤트 자동 연결 상태로 표시
                      ▶ 전시 올리기
                        ✅ 이벤트 참여 처리 완료
                        ▶ 이벤트 상세 참여작 갤러리에 노출
                          (참여작 공개 설정이 "공개"인 경우 즉시 / "비공개"인 경우 종료 후)
```
