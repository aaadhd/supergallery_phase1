## 역할 정의

| 역할 | 정의 | 비고 |
| --- | --- | --- |
| 게스트 | 비로그인 상태의 방문자 | 회원가입/로그인 없이 서비스 진입 가능 |
| 일반 창작자 | 가입 후 작품을 올리거나 감상하는 유저 | 강사 체크 OFF 상태 |
| 강사 | 프로필에서 강사 토글 ON한 창작자 | Phase 1 자율 체크. 별도 인증 없음 |
| 운영팀 | 어드민에 접근하는 내부 담당자 | 역할 기반 접근 제어 (개발자 결정) |

> 시스템(자동화 처리)은 별도 액터로 간주하며, 이 표에서는 제외.
>
