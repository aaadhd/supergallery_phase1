## 🏆 Artier's Pick

### 개념

주간 단위로 운영팀이 직접 선정하는 추천작. 선정된 작품은 메인 배너에 노출되고 'Artier's Pick' 배지가 영구 부여됩니다.

### 화면·기능 스펙

| 항목 | 내용 |
| --- | --- |
| 선정 수 | 주당 4~8점 |
| 노출 위치 | 둘러보기 상단 배너 + 갤러리 피드 상단 고정 |
| 노출 기간 | 해당 주 7일 (월~일) |
| 배지 | 작업물 카드 + 상세 모달에 'Artier's Pick' 배지 영구 표시 |
| 선정 작가 알림 | "내 작품이 이번 주 Artier's Pick에 선정됐어요!" 인앱 알림 + 이메일 발송 |
| 어드민 기능 | 작품 ID 입력 → 배지 부여 / 해제 (22번 어드민 문서 참조) |

### 선정 기준 (PD 참고 — 배지 디자인 방향)

| 기준 | 설명 |
| --- | --- |
| 작품 완성도 | 구도·색감·표현력 등 작품 자체의 퀄리티 |
| 플랫폼 정체성 부합 | Artier의 감성 — 중장년 창작자, 진정성 있는 창작 |
| 다양성 | 특정 스타일 편중 없이 다양한 작품 반영 |
| 주간 테마 연계 | 해당 주 테마와 연관된 작품 우선 고려 |

> 💡 선정 운영 절차(금요일 후보 추림 → 일요일 확정 → 월요일 발표)는 **13번 파트너 작가 운영 가이드**에서 관리합니다.
> 

### 선정 작가 혜택

- Artier's Pick 배지 영구 부여
- 메인 배너 노출 (1주)
- 공식 인스타그램 피처링
- *(Phase 2 이후)* 포인트 500 AP 자동 지급

## 🎨 주간 테마 기획전

### 개념

매주 하나의 테마를 선정하여 관련 작품을 모아 보여주는 기획전. 유저가 테마에 맞게 작품을 올릴 동기를 만들고, 갤러리에 스토리를 부여합니다.

### 화면·기능 스펙

| 항목 | 내용 |
| --- | --- |
| 발표 주기 | 매주 월요일 |
| 테마 형식 | [색상/감각] × [주제/소재] 조합 |
| 참여 방법 | 이벤트 상세 페이지에서 "참여하기" 버튼 클릭 후 업로드 |
| 노출 위치 | 이벤트 상세 페이지 참여작 갤러리 |
| 인기작 배지 | 주간 테마 참여작 중 반응 높은 작품 → '금주의 작품' 배지 |
| 어드민 기능 | 테마명 + 설명 + 기간 등록 (22번 어드민 문서 참조) |

### 테마 형식 예시

> "이번 주는 **[보라색]** 을 활용한 **[새벽 풍경]** 전시입니다"
>

## 📅 상시 캠페인

### 캠페인 1 — 내 오프라인 그림을 디지털로

| 항목 | 내용 |
| --- | --- |
| 내용 | 오프라인 그림과 그것을 디지털로 재해석한 그림을 함께 업로드 |
| 참여 방법 | 이벤트 상세 페이지에서 "참여하기" 버튼 클릭 후 업로드 (오프라인 원본 + 디지털 재해석 2장) |
| 노출 위치 | 이벤트 상세 페이지 참여작 갤러리 |

### 캠페인 2 — 나에게 영감을 준 그림 ⚠️ Phase 2

> Phase 2에서 "나도 그려봤어요" 기능과 함께 설계 예정.
> 

> 작품 상세 모달에 "나도 그려봤어요" 버튼 → 업로드 시 원작과 파생작 자동 연결.
> 

> 태그 기능 없이 버튼 하나로 참여 가능 — 시니어 친화적 구조.
> 

| 항목 | 내용 |
| --- | --- |
| 내용 | Artier 내 작품에서 영감을 받아 본인 작품 업로드 — 원작과 자동 연결 |
| 참여 방법 | 작품 상세 모달 "나도 그려봤어요" 버튼 → 업로드 플로우 진입 |
| Phase | ❌ Phase 1 미포함 — Phase 2에서 기능 설계 |

> 

| 알림 유형 | 발송 여부 | 내용 | 기본값 |
| --- | --- | --- | --- |

| 새 팔로워 | ✅ | "OO님이 팔로우하기 시작했습니다" | ON |

| 그룹전시 태그 | ✅ | "OO님이 전시에 초대했습니다" | ON |
| 팔로잉 작가 신작 | 선택 | "OO 작가가 새 작품을 올렸습니다" | OFF |
