## 1. 빈 상태 (Empty State)

> 콘텐츠가 없을 때 유저에게 보여주는 화면. 일러스트 + 안내 문구 + (필요 시) 액션 버튼 조합.
> 

| 화면 | 상황 | 한국어 문구 | 영어 문구 | 액션 버튼 |
| --- | --- | --- | --- | --- |
| 둘러보기 피드 | 콘텐츠 없음 (서비스 초기) | 아직 작품이 없어요. 조금 기다려주세요! | Works are being prepared. Check back soon! | — |
| 알림 목록 | 알림 없음 | 아직 알림이 없어요 | No notifications yet | — |
| 검색 결과 | 검색 결과 없음 | ‘{검색어}’에 대한 검색 결과가 없어요 | No results for ‘{keyword}’ | — |
| 내 프로필 — 전시 탭 | 등록한 전시 없음 | 아직 올린 전시가 없어요. 첫 작품을 올려보세요! | No exhibitions yet. Upload your first work! | 작품 올리기 |
| 내 프로필 — 작품 관리 탭 | 작품 없음 | 등록된 작품이 없어요 | No works registered yet | 작품 올리기 |
| 내 프로필 — 좋아요 탭 | 좋아요한 작품 없음 | 마음에 드는 작품에 좋아요를 눌러보세요 | Like works you enjoy to see them here | 둘러보기 |
| 내 프로필 — 저장 탭 | 저장한 작품 없음 | 저장한 작품이 없어요 | No saved works yet | 둘러보기 |
| 내 프로필 — 초안 탭 | 임시저장 없음 | 저장된 초안이 없어요 | No drafts saved | — |
| 둘러보기 피드 (개인화) | 팔로우 중인 작가 없음 | 팔로우하는 작가가 없어요. 작가를 팔로우하고 피드를 채워보세요 | No followed artists yet. Follow artists to build your feed | 둘러보기 |
| 이벤트 목록 | 참여한 이벤트 없음 | 참여한 이벤트가 없어요 | No events joined yet | 이벤트 보기 |

## 2. 에러 메시지

> 사용자 입력 오류, 시스템 오류, 일시적 실패 시 노출. 토스트 / 인라인 메시지.
> 

### 2-1. 회원가입 / 로그인

| 상황 | 한국어 문구 | 영어 문구 |
| --- | --- | --- |
| 이메일 형식 오류 | 올바른 이메일 주소를 입력해주세요 | Please enter a valid email address |
| 이미 사용 중인 이메일 | 이미 사용 중인 이메일입니다 | This email is already in use |
| 비밀번호 조건 미충족 | 영문+숫자 조합 8자 이상으로 입력해주세요 | Password must be at least 8 characters with letters and numbers |
| 이미 사용 중인 닉네임 | 이미 사용 중인 닉네임입니다 | This nickname is already taken |
| 닉네임 형식 오류 | 2자 이상 20자 이하로 입력해주세요 | Nickname must be 2–20 characters |
| 로그인 실패 | 이메일 또는 비밀번호가 일치하지 않습니다 | Incorrect email or password |
| 만 14세 미체크 | 필수 항목에 동의해야 가입할 수 있어요 | Please agree to all required terms to continue |
| 인증 메일 만료 | 인증 시간이 만료되었어요. 재발송을 요청해주세요 | The verification link has expired. Please request a new one |
| 비밀번호 재설정 링크 만료 | 링크가 만료되었어요. 재발송을 요청해주세요 | This link has expired. Please request a new one |
| 소셜 로그인 충돌 (이메일 가입이 맞다) | 이미 해당 이메일로 가입된 계정이 있어요 | An account with this email already exists |
| 소셜 로그인 충돌 (다른 소셜) | 이전에 [{OO}] 로그인으로 가입하셨습니다 | You previously signed up with [{OO}] |

### 2-2. 작품 업로드

| 상황 | 한국어 문구 | 영어 문구 |
| --- | --- | --- |
| 업로드 완료 (검수 대기) | 작품이 등록되었습니다. 운영팀 검토 후 둘러보기에 노출됩니다 (최대 24시간) | Your work has been submitted. It will appear in Browse after review (up to 24 hours) |
| 파일 형식 불일치 | JPG, PNG, WEBP, GIF 파일만 업로드할 수 있어요 | Only JPG, PNG, WEBP, or GIF files are allowed |
| 장당 최대 용량 초과 (10MB) | 각 이미지는 10MB 이하여야 해요 | Each image must be under 10MB |
| 최대 장 수 초과 (20장) | 전시당 최대 20장까지 업로드할 수 있어요 | Maximum 20 images per exhibition |

> ⚠️ 위 업로드 에러는 화면 모음 > 작품 업로드 에디터 스펙과 연계됨.

## 3. 권한 없음 (Permission Denied)

> 비로그인 상태에서 로그인 필요 액션 시도 시 보여주는 모달. 어드민 권한 없는 페이지 접근 시 403 에러.
> 

> 로그인 유도 모달 상세 스펙은 **화면 모음 > 회원가입/로그인** 참조.
> 

| 시나리오 | 한국어 모달 내용 | 영어 모달 내용 | CTA |
| --- | --- | --- | --- |
| 좋아요 시도 | 로그인하고 작품에 반응해보세요 | Log in to like this work | 로그인 / 나중에 |
| 저장 시도 | 로그인하고 작품을 저장해보세요 | Log in to save this work | 로그인 / 나중에 |
| 팔로우 시도 | 로그인하고 작가를 팔로우해보세요 | Log in to follow this artist | 로그인 / 나중에 |
| 업로드 시도 | 로그인하고 나의 작품을 올려보세요 | Log in to upload your work | 로그인 / 나중에 |
| 이벤트 참여 시도 | 로그인하고 이벤트에 참여해보세요 | Log in to join this event | 로그인 / 나중에 |
| 신고 시도 | 로그인이 필요한 기능이에요 | This feature requires login | 로그인 / 나중에 |
| 403 어드민 페이지 | 접근 권한이 없는 페이지입니다 | You don’t have permission to access this page | 홈으로 |

> "나중에" 클릭 시: 모달 닫고 현재 화면 유지. "로그인" 클릭 시: 로그인 화면으로 이동 (redirect 파라미터 포함).
