## 탈퇴 후 데이터 처리 원칙

| 항목 | 정책 |
| --- | --- |
| 작품 처리 | 익명 유지 ("작가 미상"으로 표기). 전시에서 제거하지 않음 |
| username 유지 | 작품이 남기 때문에 username 삭제 안 함 |
| 닉네임(displayName) | **영구 잠금** — 탈퇴 후 작가명은 **"작가 미상"** (EN: **"Unknown Artist"**) 으로 대체된다. "탈퇴" 등 부정적 뉴앙스를 피하고, 미술관에서 작자 불명 시 쓰는 관용적 표현을 차용.
> 작품 노출 상세 정책 (피드/검색/좋아요/클릭 동작 등)은 **기능 모음 > 회원 탈퇴 > 탈퇴 유저 작품 노출 정책** 참조. (닉네임 = 필명 보호) |
| 이메일 | 주소 유지 (Social 계정 연결 해제 제외) |
| 개인정보 (좋아요·팔로우·프로필 등) | 즉시 삭제 후 복구 불가. 작품은 익명 유지되므로 제외 |

## 탈퇴 프로세스

```jsx
계정 설정 > 탈퇴 진입
  ↓
탈퇴 사유 선택
  ↓
탈퇴 안내 모달 + 비밀번호 or 소셜 인증 재확인
  ↓
"탈퇴하기" 버튼 클릭 → 탈퇴 처리 완료
  ↓
탈퇴 완료 이메일 자동 발송
  ↓
로그아웃 처리 → 둘러보기 페이지 이동
  ↓
세션 일괄 만료
```

> 탈퇴 사유: 서비스가 마음에 들지 않음 / 원하는 콘텐츠를 찾기 어려움 / 이용 빈도가 낮아짐 / 기타
> 

> 상세 플로우는 **기능 모음 > 회원 탈퇴** 참조.
> 

## 닉네임 유니크(Unique) 정책

> **설계 배경**: username과 displayName이 분리되어 있으나, 사용자가 displayName 기반으로 활동할 것으로 예상되는 서비스 특성상 displayName도 Unique 정책 유지.
> 
- displayName Unique 적용 → 다른 유저가 탈퇴한 닉네임과 동일한 닉네임으로 가입 불가
- **탈퇴 후 30일이 지나도 해당 닉네임 사용 불가**

| 탈퇴 완료 | "탈퇴가 완료되었습니다." (+ 탈퇴 완료 이메일 발송) |
