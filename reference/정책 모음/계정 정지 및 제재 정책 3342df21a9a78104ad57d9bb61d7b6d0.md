# 계정 정지 및 제재 정책

관련 화면: 어드민 (관리자 기능), 계정 설정
구분: 운영 정책

> 어뷰징 유저 제재 기준 및 계정 정지 데이터 처리 정책.
> 

> 탈퇴 후 데이터 처리는 **정책 모음 > 회원 탈퇴 처리 정책** 참조.
> 

---

## 제재 단계

| 단계 | 제재 | 조건 |
| --- | --- | --- |
| 1단계 주의 | 경고 알림 (이메일 + 인앱) | 콘텐츠/신고 정책 1회 위반 |
| 2단계 정지 | **7일 기간 정지** | 경고 후 반복 위반 or 중대 위반 |
| 3단계 정지 | **30일 기간 정지** | 2단계 후 재발 |
| 4단계 정지 | **영구 정지** | 반복적 중대 위반 or 불법 콘텐츠 |

> 단계 없이 **즉시 영구 정지** 가능한 경우: 학대 폭력·사지 오용·실제 피해 유발 중수 위반
> 

## 정지 주요 사유

- 콘텐츠/신고 정책 반복 위반
- 탈취를 위한 다중 계정 사용
- 스팸/광고 콘텐츠 대량 업로드
- 다른 유저를 타깃으로 한 괴롭힘 또는 지속적인 스토킹

## 정지 중 계정 동작

| 항목 | 내용 |
| --- | --- |
| 로그인 시도 | "계정이 정지되었습니다" 안내 + 사유 + 만료 날짜 표시 |
| 기존 콘텐츠 | 정상 노출 유지 (탈퇴 아님) |
| 알림 | 계정 정지 안내 이메일 자동 발송 |
| 이의제기 | 문의하기 폼으로 접수 (Phase 1) |

## 영구 정지 (Permanent Ban)

- 영구 정지된 이메일로 재가입 불가
- 영구 정지 이력 DB 별도 기록
