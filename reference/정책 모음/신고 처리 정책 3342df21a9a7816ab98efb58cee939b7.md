## 블라인드 처리 기준

> 신고 접수만으로 자동 블라인드 되지 않음 — 운영팀 검토 후 처리 결정.
> 
- 동일 유저가 동일 대상에 중복 신고 불가 (시스템 차단)
- **1주일 이내 허위 신고 3회 이상** 확인 시 → 신고자에게 경고
- 경고 후 지속 시 → **7일 신고 차단**

> 기능 모음 > 신고 처리 플로우와 동일 기준.
> 

## 신고자 통보 정책

- **신고 접수 시**: 토스트로 즉시 확인 — "신고가 접수되었습니다" (EN: "Your report has been submitted")
- **신고 처리 결과**: 신고자에게 **별도 안내 없음** (단, 삭제 시 이의제기 연락처 제공)
- 이의제기: Phase 2에서 공식 DMCA 페이지 도입 예정. Phase 1은 단순 이메일 접수 방식.
