Figma Make로 데스크탑 웹(1440px) MVP를 한 번에 만들어줘. Behance의 메인 내비/구조를 최대한 그대로 참고하되, 우리 서비스는 “디지털 드로잉 갤러리 + 전시룸 + 커뮤니티 + 거래(소장) + 러닝(클래스)”를 포함한다. 시니어 특화 UX는 고려하지 않아도 된다(일반 웹 UX로 깔끔하게).

[핵심 원칙]
- Shop(소장/거래)은 GNB로 따로 빼지 말고 “둘러보기(Discover)” 안에 묶는다.
- Discover 내부에서 “감상(전체)”과 “구매 가능만”을 토글/탭으로 전환한다.
- Behance처럼 작품(프로젝트) 카드 그리드가 중심이고, 프로필/팔로우/좋아요/댓글/저장(컬렉션)이 커뮤니티의 핵심이다.
- 전시룸(Room)은 개인이 꾸미는 전시 공간이며, 전시룸 페이지 안에서 커뮤니티(방명록/Q&A)가 허브가 되게 한다.
- 러닝(Learn)은 GNB에 별도 메뉴로 둔다(클래스 허브). 작품/작가/전시룸과 연결되는 구조로 만든다.
- 의뢰(커미션) 메뉴/페이지는 만들지 않는다.

[브랜드/스타일]
- 톤: 모던 미술 갤러리 + 편집샵 감성(작품이 주인공)
- 언어: 한국어 UI 카피
- 폰트: Pretendard(또는 Inter + Pretendard)
- 컬러: 배경 밝은 톤(화이트/오프화이트), 텍스트 다크, 포인트 컬러 1개(블루 또는 골드 계열 아무거나)로 통일
- 레이아웃: 카드 그리드 중심, 여백 넉넉, 미니멀 아이콘

[GNB(Behance 구조 유사)]
- 로고(좌)
- 메뉴: 둘러보기 / 전시룸 / 러닝
- 우측: 검색(통합) / 알림 / 메시지(선택) / 프로필 아바타
- 주요 액션 버튼 2개:
  1) “작품 공유”(Create/Upload)
  2) “지금 구매”(Purchase CTA) — 단, 이 버튼은 ‘둘러보기 > 구매 가능’ 모드로 이동하는 숏컷이다(별도 Shop 섹션은 만들지 않음)

[필수 엔티티/콘텐츠 타입]
- 작품(Work): 디지털 드로잉 이미지, 제목, 작가, 좋아요, 저장, 댓글
- 전시룸(Room): 커버, 제목, 소개, 작품 모음, 공유
- 클래스(Class): 제목, 강사(작가), 난이도, 가격, 커리큘럼, 질문(커뮤니티)
- 거래 타입(작품의 소장 옵션): 다운로드 / 프린트 / 라이선스 (옵션 선택 UI만 구현)

[사이트맵 + 생성할 화면(프레임명 그대로 생성)]
1) WEB-1440 / Discover / Home
- 상단: 검색바
- Discover 상단 세그먼트/탭: “감상(전체)” / “구매 가능만”
- 섹션: “추천 전시룸” (Room 카드 4개)
- 섹션: “인기 작품” (Work 카드 8~12개 그리드)
- 섹션: “새로 올라온 작품” (Work 카드 그리드)
- 섹션: “러닝 추천 클래스” (Class 카드 4개)
- 섹션: “팔로잉 피드” (작가 업데이트형 카드 3개)

2) WEB-1440 / Discover / Browse(Works Grid)
- 상단: 토글 “감상(전체)” / “구매 가능만”
- 정렬: 최신/인기/트렌딩
- 그리드: 작품 카드 4~5열
- “구매 가능만” 모드에서는 카드에 작은 “소장하기” 배지/버튼이 추가됨

3) WEB-1440 / Works / Detail
- 좌: 큰 작품 이미지 + 썸네일(선택)
- 우: 제목/작가/좋아요/저장/공유
- 댓글 섹션(기본 커뮤니티)
- “소장하기” 버튼(조용한 CTA) → 클릭 시 하단/우측 드로어가 열리며
  - 소장 옵션: 다운로드/프린트/라이선스 선택
  - 프린트는 사이즈 선택(A3/A2)
  - 라이선스는 용도 선택(출판/굿즈/브랜딩) + 문의 버튼
- 하단: “이 작가의 다른 작품”, “이 작품이 들어있는 전시룸”

4) WEB-1440 / Rooms / Directory
- 전시룸 카드 그리드 3열
- 정렬: 인기/최신
- 검색 가능

5) WEB-1440 / Rooms / Detail (전시룸 = 커뮤니티 허브)
- 상단: 커버 + 제목 + 소개 + 작가 카드 + 공유 버튼
- 탭 4개 크게:
  1) 전시(Works) — 벽면 보기 느낌의 그리드(큰 썸네일+여백)
  2) 방명록 — 짧은 코멘트 리스트 + 작성창
  3) Q&A — 질문 리스트(작품 첨부 가능) + 질문 작성 버튼
  4) 소장하기 — 전시룸 내 구매 가능 작품만 모아보기(번들 느낌, 장바구니 없이도 됨)
- 전시 탭에서 작품 클릭 시 라이트박스 감상 모드로 열림(닫기/이전/다음/확대)

6) WEB-1440 / Rooms / Create(Template Select)
- 템플릿 카드 3개:
  - 포트폴리오 전시
  - 테마 전시
  - 리턴전(재데뷔전)
- 선택 후 편집 화면으로 이동

7) WEB-1440 / Rooms / Edit(Room Builder)
- 좌 패널: 제목/소개/커버/레이아웃 선택(그리드/벽면/리스트)
- 우 패널: 미리보기
- “작품 추가” 모달: 내 작품 목록에서 다중 선택
- 상단 CTA: “공개/공유”

8) WEB-1440 / Learn / Home
- 섹션: 추천 클래스 / 인기 클래스 / 신규 클래스
- 카테고리 칩: 기초/색감/인물/풍경/툴팁(임의)
- 클래스 카드 그리드

9) WEB-1440 / Learn / Class Detail
- 클래스 헤더: 제목/강사/가격/수강 신청 버튼
- 커리큘럼(세션 리스트)
- “질문 게시판(Q&A)” 탭(클래스 전용 Q&A)
- 강사(작가) 작품/전시룸 링크

10) WEB-1440 / Create / Upload(3-step)
- Step 1: 파일 업로드(드래그앤드롭)
- Step 2: 제목(필수), 설명(선택), 태그(스타일/테마 위주, 선택)
- Step 3: 공개 설정(공개/링크공유)
- 완료 화면: “업로드 완료” + CTA “전시룸에 담기” / “둘러보기로 가기”

11) WEB-1440 / Profile / My Profile
- 헤더: 프로필/닉네임/한줄소개 + 팔로워/팔로잉
- 탭: 작품 / 전시룸 / 클래스 / 저장(컬렉션) / 활동(댓글·Q&A)
- “작품 공유” 버튼

12) WEB-1440 / Search / Results
- 탭: 전체 / 작품 / 전시룸 / 작가 / 클래스 / Q&A
- 결과 리스트/그리드

[프로토타입 연결(필수)]
- GNB “둘러보기” → Discover/Home
- Discover에서 작품 카드 클릭 → Works/Detail
- Works/Detail에서 “소장하기” → 옵션 드로어 열기
- Discover에서 전시룸 카드 클릭 → Rooms/Detail
- Rooms/Detail에서 작품 클릭 → 라이트박스
- GNB “러닝” → Learn/Home → Class Detail
- “작품 공유” → Create/Upload steps → 완료 → Works/Detail
- 프로필 진입 → Profile/My Profile

[더미 데이터]
- 작가: 이선화, 김영자, 박정호, 최미경, 정은수
- 전시룸: 봄빛의 기억, 고요한 오후, 라인의 미학, 리턴전 2026
- 작품: 창가의 빛, 바람의 결, 분당의 오후, 오래된 정원
- 클래스: 디지털 수채의 색감, 인물 명암 빠르게 잡기, 라인드로잉의 리듬
- 좋아요 수는 다양하게

[추가 요구]
- 디자인 시스템 컴포넌트 프레임도 생성:
  Buttons, Cards(Work/Room/Class), Tabs, Modals/Drawers(소장하기 드로어, 작품 라이트박스), Forms(업로드/Q&A)
- 전반적으로 Behance 메인 구조를 닮게 하되, Shop은 Discover 안 “구매 가능만” 모드로 흡수되어 보이게 해줘.
