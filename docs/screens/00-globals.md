# 00. 공통 요소 — 헤더 · 푸터 · 하단 네비게이션

> 모든 일반 화면(어드민·집중 플로우 제외)에 공통으로 노출되는 글로벌 영역. 이 문서가 비어 있던 부분 보강.

---

## 사전 지식

### 어떤 화면에 노출되나

| 영역 | 노출되는 화면 | 노출 안 되는 화면 |
|---|---|---|
| 헤더 + 푸터 + 모바일 하단 GNB | 둘러보기·검색·프로필·전시·이벤트·알림·설정·정보 페이지 등 일반 사용자 화면 | 로그인·가입·온보딩·비밀번호 재설정·점검 (집중 플로우) / 어드민 콘솔 (자체 레이아웃) |

### 시니어 친화 정책 (전 영역 적용)

- 모든 인터랙티브 영역 최소 **44×44 픽셀**
- 헤더 아이콘 사이 충분한 간격 (실수 탭 방지)
- 모바일 하단 GNB는 큰 아이콘 + 라벨 텍스트 항상 표시 (아이콘만 있는 디자인 X)

## SCR-GLOBAL-01 · 헤더 (Header / GNB)

| 항목 | 값 |
|---|---|
| 노출 위치 | 일반 사용자 화면 상단 고정 |
| 다국어 | 한국어·영어 |

### 화면의 목적
어디서든 **검색·알림·프로필·작품 올리기**에 한 번에 도달. 시니어가 헤매지 않도록 항상 같은 위치에 배치.

### 데스크톱 헤더 구성 (왼쪽 → 오른쪽)

| 영역 | 요소 | 비고 |
|---|---|---|
| 로고 | "Artier" 워드마크 + 작은 로고 | 탭 시 둘러보기로 |
| 메인 메뉴 | 둘러보기 · 이벤트 | 현재 화면 활성 표시 |
| 검색 아이콘 | 돋보기 | 탭 시 검색 화면으로 |
| **[작품 올리기]** | 강조 버튼 (primary) | 로그인 필수 — 비로그인이면 로그인 유도 |
| 알림 벨 | 종 + **안 읽음 카운트 배지** (>0일 때만) | 로그인 시만 표시 |
| 아바타 | 본인 프로필 이미지 (없으면 이니셜) | 탭 시 드롭다운 |
| (비로그인) 로그인 버튼 | 아바타 위치에 [로그인] | |

### 모바일 헤더 구성

데스크톱과 동일하되 **메인 메뉴(둘러보기·이벤트)는 하단 GNB로 이동**, 헤더에는 로고·검색·알림·아바타만.

### 아바타 드롭다운 메뉴 (로그인 시)

| 메뉴 | 이동 |
|---|---|
| 내 프로필 | 본인 프로필 |
| 작품 올리기 | 업로드 |
| 알림 | 알림 화면 |
| 설정 | 설정 화면 |
| 운영 콘솔 | 어드민 (운영팀 권한 있을 때만 노출) |
| 로그아웃 | 로그아웃 처리 후 둘러보기로 |

### 탭·액션 결과 (네비게이션)

| 탭 대상 | 결과 |
|---|---|
| 로고 / "Artier" | **둘러보기(`/`)** 로 이동 |
| "둘러보기" 메뉴 | 둘러보기로 |
| "이벤트" 메뉴 | 이벤트 목록(`/events`)으로 |
| 검색 아이콘 | **검색(`/search`)** 으로 |
| **[작품 올리기]** | 로그인 시 → 업로드(`/upload`). 비로그인 시 → 로그인 유도 모달 → [로그인] 시 `/login?redirect=/upload` |
| 알림 벨 | **알림(`/notifications`)** 으로 |
| 아바타 | 같은 화면 위에 드롭다운 열림 |
| (비로그인) [로그인] | **로그인(`/login`)** 으로. 현재 화면 URL이 `redirect`로 전달 |
| 드롭다운 항목 — 내 프로필 | `/profile` 또는 `/me` |
| 드롭다운 항목 — 알림 / 설정 | 각각 `/notifications`, `/settings` |
| 드롭다운 항목 — 운영 콘솔 | `/admin` (권한 있는 사용자만 메뉴 표시) |
| 드롭다운 항목 — 로그아웃 | 로그아웃 처리 → 둘러보기로 |

### 상태·분기

| 상태 | 표현 |
|---|---|
| 비로그인 | 알림 벨·아바타 숨김. 우측에 [로그인] 버튼 |
| 로그인 + 안 읽음 알림 | 벨 위에 빨간 카운트 배지 (>99이면 "99+") |
| 정지 상태 | [작품 올리기] 등 일부 액션 비활성. 드롭다운에 정지 안내 표시 |
| 어드민 권한 | 드롭다운에 "운영 콘솔" 메뉴 추가 |

### Phase 2 보강
- 검색 자동완성 인라인 노출 (헤더 안에 펼쳐지는 형태)
- 알림 카운트 실시간 업데이트 (현재는 화면 진입 시 갱신)
- 다크 모드 토글

## SCR-GLOBAL-02 · 푸터

| 항목 | 값 |
|---|---|
| 노출 위치 | 일반 사용자 화면 하단 |

### 화면의 목적
법적 의무 링크 + 운영 정보 + 언어 선택. 데스크톱에서 항상 노출, 모바일에서는 길게 스크롤한 끝에 보임.

### 푸터 구성

| 영역 | 요소 |
|---|---|
| 브랜드 영역 | 로고 + "시니어 작가의 디지털 갤러리" 짧은 소개 |
| 서비스 링크 | 둘러보기 · 이벤트 · 작품 올리기 |
| 정보 링크 | 서비스 소개 · FAQ · 공지사항 · 문의 |
| 법적 링크 | 이용약관 · 개인정보처리방침 |
| 연락처 | "문의: [고객지원 이메일]" — 메일 링크 |
| 언어 토글 | 한국어 / English |
| 카피라이트 | "© 2026 Artier" |

### 탭·액션 결과 (네비게이션)

| 탭 대상 | 결과 |
|---|---|
| 로고 | 둘러보기(`/`)로 |
| 서비스 링크 — 둘러보기 / 이벤트 / 작품 올리기 | 각각 `/` · `/events` · `/upload` (업로드는 비로그인 시 로그인 유도) |
| 정보 링크 — 서비스 소개 / FAQ / 공지사항 / 문의 | 각각 `/about` · `/faq` · `/notices` · `/contact` |
| 법적 링크 — 이용약관 / 개인정보처리방침 | 각각 `/terms` · `/privacy` |
| 고객지원 이메일 | 기기 메일 클라이언트 열림 (mailto) |
| 언어 토글 | 같은 화면에서 즉시 언어 전환 |

### Phase 2 보강
- SNS 링크 (인스타·유튜브)
- 사업자 정보 (실서비스 의무)
- 채용·블로그 등

## SCR-GLOBAL-03 · 모바일 하단 네비게이션 (Bottom GNB)

| 항목 | 값 |
|---|---|
| 노출 위치 | 모바일 화면 하단 고정 (sm 이하) |
| 노출 안 되는 화면 | 집중 플로우(로그인·가입·온보딩 등), 어드민, 일부 모달 풀스크린 상태 |

### 화면의 목적
모바일에서 핵심 4~5개 메뉴를 한 손 엄지 닿는 위치에. 시니어가 메뉴 위치를 외울 필요 없도록 큰 아이콘 + 라벨.

### 메뉴 구성 (4~5개)

| 아이콘 | 라벨 | 이동 |
|---|---|---|
| 홈 | 둘러보기 | `/` |
| 돋보기 | 검색 | `/search` |
| 플러스 (강조) | 작품 올리기 | `/upload` (비로그인이면 로그인 유도) |
| 달력 | 이벤트 | `/events` |
| 사람 | 내 프로필 | `/profile` (비로그인이면 로그인 유도) |

### 디자인 원칙
- 모든 항목 **아이콘 + 텍스트 라벨** 동시 표시 (아이콘만은 시니어에게 모호함)
- 현재 화면 항목은 색상 강조
- 가운데 "작품 올리기"는 약간 더 크게 / 강조색으로 핵심 액션임을 시각화

### 탭·액션 결과 (네비게이션)

| 탭 대상 | 결과 |
|---|---|
| 홈 | 둘러보기로 |
| 검색 | 검색으로 |
| 작품 올리기 (강조) | 업로드로 (비로그인 시 로그인 유도 모달) |
| 이벤트 | 이벤트 목록으로 |
| 내 프로필 | 본인 프로필로 (비로그인 시 로그인 유도 모달) |

### 상태·분기

| 상태 | 표현 |
|---|---|
| 비로그인 + "내 프로필"·"작품 올리기" 탭 | 로그인 유도 모달 (세션 1회 정책) |
| 정지 | "작품 올리기" 비활성 |
| 키보드 노출 (모바일 입력 중) | GNB 자동 숨김 |

## 공통 부가 요소

### 토스트 (sonner)

| 항목 | 정책 |
|---|---|
| 위치 | 화면 상단 중앙 |
| 지속 시간 | 5초 |
| 중복 | 같은 메시지 연속 시 합치지 않음 (각각 표시 — 사용자 행동 피드백 명확) |
| 색상 | success(녹) · error(빨) · info(중립) |

### 스플래시 화면

앱 최초 로드 시 짧게 노출되는 로딩 화면. 한 번 본 사용자는 다음 진입에서 생략(localStorage 플래그). 개발/테스트는 강제 노출 옵션.

### 오프라인 배너

네트워크 끊김 감지 시 화면 상단에 안내 배너 ("인터넷 연결을 확인해주세요"). 복구 시 자동 사라짐.

### 스크롤 위치 복원

둘러보기·검색 결과·프로필 등 리스트 화면은 **이탈 시 스크롤 위치 저장 → 재진입 시 복원**. 시니어가 스크롤 다시 내릴 부담을 덜기 위함.
