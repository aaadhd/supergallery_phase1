# 00. 공통 요소 — 헤더 · 푸터 · 하단 네비게이션

> 모든 일반 화면(어드민·집중 플로우 제외)에 공통으로 노출되는 글로벌 영역. 이 문서가 비어 있던 부분 보강.

---

## 사전 지식

### 어떤 화면에 노출되나

| 영역 | 노출되는 화면 | 노출 안 되는 화면 |
|---|---|---|
| 헤더 + 푸터 + 모바일 하단 GNB | 둘러보기·검색·프로필·전시·이벤트·알림·설정·정보 페이지 등 일반 사용자 화면 | 로그인·가입·온보딩·비밀번호 재설정·점검 (집중 플로우) / 어드민 콘솔 (자체 레이아웃) |

### 시니어 친화 정책 (전 영역 적용)

- 모든 인터랙티브 영역 최소 **44×44 픽셀**
- 헤더 아이콘 사이 충분한 간격 (실수 탭 방지)
- 모바일 하단 GNB는 큰 아이콘 + 라벨 텍스트 항상 표시 (아이콘만 있는 디자인 X)

## SCR-GLOBAL-01 · 헤더 (Header / GNB)

| 항목 | 값 |
|---|---|
| 노출 위치 | 일반 사용자 화면 상단 고정 |
| 다국어 | 한국어·영어 |

### 화면의 목적
어디서든 **검색·알림·프로필·작품 올리기**에 한 번에 도달. 시니어가 헤매지 않도록 항상 같은 위치에 배치.

### 데스크톱 헤더 구성 (왼쪽 → 오른쪽)

| 영역 | 요소 | 비고 |
|---|---|---|
| 로고 | 로고 이미지 + "Artier" 브랜드명 | 탭 시 둘러보기로 |
| 메인 메뉴 | 둘러보기 · 이벤트 | 현재 화면 활성 표시 (`md` 이상에서만 표시) |
| **[작품 올리기]** | 강조 버튼 (primary, rounded-full) | **로그인 시에만 표시** |
| 아바타 | 본인 프로필 이미지 (없으면 이니셜 fallback) | 탭 시 `/me`로 직접 이동 (드롭다운 없음) |
| 검색 아이콘 | 돋보기 | 탭 시 검색 화면으로 |
| 알림 벨 | 종 + **안 읽음 카운트 배지** (>0일 때만) | 로그인 시만 표시 |
| 설정 아이콘 | 톱니바퀴 (Settings) | 탭 시 `/settings`로. 로그인 시만 표시 |
| 언어 선택 | 지구본 (Globe) 드롭다운 — 한국어 / English | `sm` 이상에서 표시. 로그인 여부 무관 |
| (비로그인) [로그인] 버튼 | [작품 올리기] 위치에 [로그인] | `auth.login()` 데모 즉시 로그인 |
| (비로그인) 검색 아이콘 | 돋보기 | 비로그인에서도 검색 가능 |

> **참고**: 아바타는 드롭다운 메뉴가 아니라 `/me`로 직접 이동하는 단일 버튼이다. 로그아웃은 설정 화면(`/settings`)에서만 가능.

### 모바일 헤더 구성

데스크톱과 달리 **메인 메뉴(둘러보기·이벤트)·검색·업로드·아바타는 하단 GNB로 이동**. 모바일 헤더에는 **로고 + 브랜드명, 알림 벨, 설정 톱니바퀴, 언어 선택 지구본**만 노출.

### 탭·액션 결과 (네비게이션)

| 탭 대상 | 결과 |
|---|---|
| 로고 / "Artier" | **둘러보기(`/`)** 로 이동 |
| "둘러보기" 메뉴 | 둘러보기로 |
| "이벤트" 메뉴 | 이벤트 목록(`/events`)으로 |
| 검색 아이콘 | **검색(`/search`)** 으로 |
| **[작품 올리기]** | 업로드(`/upload`)로. 이미 `/upload`이면 `?new=<timestamp>` 쿼리로 새 플로우 시작. **비로그인 시 버튼 자체가 미노출** (대신 [로그인] 버튼) |
| 알림 벨 | **알림(`/notifications`)** 으로 |
| 아바타 | **프로필(`/me`)** 로 직접 이동 |
| (비로그인) [로그인] | `auth.login()` 즉시 데모 로그인 실행 + `persistMockSession`. 온보딩 미완이면 `/onboarding`으로 이동 |
| 설정 아이콘 | **설정(`/settings`)** 으로 |
| 언어 지구본 | 드롭다운에서 한국어/English 선택 → 즉시 전환 |

### 상태·분기

| 상태 | 표현 |
|---|---|
| 비로그인 | 알림 벨·아바타·설정·[작품 올리기] 숨김. 우측에 [로그인] 버튼 + 검색 아이콘 |
| 로그인 + 안 읽음 알림 | 벨 위에 빨간 카운트 배지 (>99이면 "99+") |

### Phase 2 보강
- 검색 자동완성 인라인 노출 (헤더 안에 펼쳐지는 형태)
- 알림 카운트 실시간 업데이트 (현재는 화면 진입 시 갱신)
- 다크 모드 토글

## SCR-GLOBAL-02 · 푸터

| 항목 | 값 |
|---|---|
| 노출 위치 | 일반 사용자 화면 하단 |

### 화면의 목적
법적 의무 링크 + 운영 정보. 데스크톱에서 항상 노출, 모바일에서는 스크롤 끝에 보임. 업로드 화면(`/upload`)에서는 푸터 숨김.

### 푸터 구성

한 줄짜리 하단 바 + 그 위로 펼쳐지는 사업자 정보 패널. 노트폴리오 스타일.

#### 하단 바 (항상 노출)

| 요소 | 비고 |
|---|---|
| **사업자 정보** 토글 버튼 | 탭 시 위로 상세 패널 펼침/접힘. 화살표 아이콘 동반 |
| 이용약관 | `/terms` |
| 개인정보처리방침 | `/privacy` |
| 문의 | `/contact` |
| 공지사항 | `/notices` (`sm` 이상에서만 표시) |
| FAQ | `/faq` (`sm` 이상에서만 표시) |
| 서비스 소개 | `/about` (`md` 이상에서만 표시) |
| 쿠키 설정 | 쿠키 동의 초기화 버튼 (`md` 이상에서만 표시). `artier_cookie_consent` 삭제 후 배너 재노출 |
| 카피라이트 | "© {year} Artier" (`sm` 이상에서만 표시, 우측 정렬) |

#### 사업자 정보 패널 (펼침 시)

"사업자 정보" 버튼 탭 시 바 위로 슬라이드 업되는 상세 패널:
- 회사명 · 대표자 · 사업자등록번호
- 통신판매업 신고번호 · 개인정보보호책임자
- 주소 · 이메일 · 전화번호
- 카피라이트 · 이메일 무단수집 거부 문구 · 관할 법원

외부 클릭 또는 `Esc` 키로 닫힘.

> **참고**: 언어 전환은 푸터가 아닌 헤더의 Globe 드롭다운에서만 제공. 푸터에 언어 토글 없음.

### 탭·액션 결과 (네비게이션)

| 탭 대상 | 결과 |
|---|---|
| 이용약관 | `/terms` |
| 개인정보처리방침 | `/privacy` |
| 문의 | `/contact` |
| 공지사항 | `/notices` |
| FAQ | `/faq` |
| 서비스 소개 | `/about` |
| 쿠키 설정 | `artier_cookie_consent` 삭제 → 쿠키 동의 배너 재표시 |
| 사업자 정보 토글 | 사업자 정보 패널 열기/닫기 |

### Phase 2 보강
- SNS 링크 (인스타·유튜브)
- 채용·블로그 등

## SCR-GLOBAL-03 · 모바일 하단 네비게이션 (Bottom GNB)

| 항목 | 값 |
|---|---|
| 노출 위치 | 모바일 화면 하단 고정 (`md` 미만, 즉 768px 미만. `md:hidden`) |
| 노출 안 되는 화면 | 집중 플로우(로그인·가입·온보딩 등), 어드민, 일부 모달 풀스크린 상태 |

### 화면의 목적
모바일에서 핵심 4~5개 메뉴를 한 손 엄지 닿는 위치에. 시니어가 메뉴 위치를 외울 필요 없도록 큰 아이콘 + 라벨.

### 메뉴 구성 (5개, 순서 주의)

| 순서 | 아이콘 | 라벨 | 이동 |
|---|---|---|---|
| 1 | Home | 둘러보기 | `/` |
| 2 | CalendarDays | 이벤트 | `/events` |
| 3 | **Plus (강조 원형)** | 올리기 | `/upload` (비로그인이면 `LoginPromptModal`) |
| 4 | Search | 검색 | `/search` |
| 5 | User | 프로필 / MY | `/me` (비로그인이면 `LoginPromptModal`) |

### 디자인 원칙
- 모든 항목 **아이콘 + 텍스트 라벨** 동시 표시 (아이콘만은 시니어에게 모호함)
- 현재 화면 항목은 색상 강조
- 가운데 "작품 올리기"는 약간 더 크게 / 강조색으로 핵심 액션임을 시각화

### 탭·액션 결과 (네비게이션)

| 탭 대상 | 결과 |
|---|---|
| 둘러보기 | `/`로 |
| 이벤트 | `/events`로 |
| 올리기 (강조) | 로그인 시 → `/upload`. 비로그인 시 → `LoginPromptModal` (데모 즉시 로그인 실행) |
| 검색 | `/search`로 |
| 프로필 / MY | 로그인 시 → `/me`. 비로그인 시 → `LoginPromptModal` |

### 상태·분기

| 상태 | 표현 |
|---|---|
| 비로그인 + "프로필"·"올리기" 탭 | `LoginPromptModal` 표시. 모달 내 [로그인] 클릭 시 `authStore.login()` 즉시 실행 |
| 정지 상태에서 업로드 차단 | 미구현 — 헤더·GNB에 정지 검사 없음 |
| 키보드 노출 시 GNB 자동 숨김 | 미구현 — 키보드 감지 로직 없음 |

## 공통 부가 요소

### 토스트 (sonner)

| 항목 | 정책 |
|---|---|
| 위치 | 화면 상단 중앙 |
| 지속 시간 | 5초 |
| 중복 | 같은 메시지 연속 시 합치지 않음 (각각 표시 — 사용자 행동 피드백 명확) |
| 색상 | success(녹) · error(빨) · info(중립) |

### 스플래시 화면

앱 최초 로드 시 짧게 노출되는 로딩 화면 (기본 1800ms). 한 번 본 사용자는 같은 세션 내 재진입 시 생략(**sessionStorage** `artier_splash_seen` 플래그 — 탭/창 닫으면 초기화).

### 오프라인 배너

네트워크 끊김 감지 시 화면 상단에 안내 배너 ("인터넷 연결을 확인해주세요"). 복구 시 자동 사라짐.

### 스크롤 위치 복원

**둘러보기(`Browse`)** 화면만 이탈 시 스크롤 위치 저장 → 재진입 시 복원 (`scrollRestore.ts`, sessionStorage `artier_scroll_` 키). 검색·프로필 등 다른 리스트 화면은 미구현.

### 쿠키 동의 배너

`CookieConsent` 컴포넌트. 앱 최초 진입 시 하단에 쿠키 동의 배너 노출. 동의 시 `artier_cookie_consent` localStorage에 저장. 푸터의 "쿠키 설정" 버튼으로 동의 초기화 가능.

### 글꼴 크기 설정

`fontScale.ts` + 설정 화면(`/settings`), 온보딩에서 글꼴 크기 조절 가능. `artier_font_scale` localStorage 키. `html` 요소의 `font-size`에 반영.

### 본문 바로가기 (Skip to content)

`Layout.tsx`에서 `<a href="#main-content">` 형태의 접근성 스킵 링크 제공. 평소 `sr-only`, 포커스 시 화면에 노출.

### QA 바로가기 플로팅 버튼

`QaScreenShortcuts` 컴포넌트. DEV 모드 또는 `VITE_FOOTER_QA_LINKS` 환경변수 활성 시 화면 하단에 검수용 플로팅 버튼 노출.
