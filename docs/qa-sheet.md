## 1. 작품 올리기 (Upload)

### 1.1 개인 전시 — 기본 발행

| # | 단계 | 기대 결과 | Pass |
|---|---|---|---|
| 1 | `/upload` 진입 (로그인 상태) | 업로드 화면 표시, 유형 선택(개인/그룹) | ☐ |
| 2 | "개인 전시" 선택 → 이미지 1장 추가 | 이미지 카드 표시, 썸네일 렌더 | ☐ |
| 3 | 전시명 입력 (20자 이내) | 글자수 카운터 표시, 20자 초과 차단 | ☐ |
| 4 | 원본 확인 체크 → [발행하기] | 세부정보 모달(작품명·커버 선택) 열림 | ☐ |
| 5 | 세부정보 모달에서 [완료] | 발행 처리 → 프로필 전시 탭으로 이동 | ☐ |
| 6 | 프로필 전시 탭 확인 | 작품 카드에 "검수 중" 배지 표시 | ☐ |
| 7 | 상단 배너 확인 | "검수 후 공개돼요" 배너 표시, 닫기(×) 가능 | ☐ |
| 8 | 둘러보기 피드 확인 | 해당 작품 **미노출** | ☐ |

### 1.2 개인 전시 — 다장 이미지

| # | 단계 | 기대 결과 | Pass |
|---|---|---|---|
| 1 | 이미지 3장 추가 | 3개 카드, 드래그 순서 변경 가능 | ☐ |
| 2 | 각 이미지에 작품명 입력 | 개별 작품명 저장 | ☐ |
| 3 | 세부정보 모달에서 커버 이미지 선택 | 선택한 이미지가 카드 썸네일에 반영 | ☐ |
| 4 | 발행 후 프로필 확인 | 카드에 이미지 개수 배지 "📷 3" 표시 | ☐ |

### 1.3 그룹 전시 — 기본 발행

| # | 단계 | 기대 결과 | Pass |
|---|---|---|---|
| 1 | "그룹 전시" 선택 | 그룹명 입력란 + 작가 지정 영역 표시 | ☐ |
| 2 | 그룹명 입력 | 자동완성 후보 드롭다운 표시 | ☐ |
| 3 | 이미지별 작가 지정 — 회원 검색 | 검색 결과에서 선택 가능 | ☐ |
| 4 | 이미지별 작가 지정 — 비회원 직접 입력 | 이름 + 전화번호 입력란 표시 | ☐ |
| 5 | 발행 → 비회원 초대 확인 모달 | 수신자 목록 + 실명 안내 경고 표시 | ☐ |
| 6 | [발송] 누르기 | 발행 완료, SMS 모의 발송 로그 기록 | ☐ |
| 7 | 프로필 전시 탭 확인 | 그룹명 배지 표시 | ☐ |

### 1.4 강사 업로드

## 7. 역할별 화면 분기

### 7.1 프로필 — 역할별 차이

| # | 역할 | 기대 결과 | Pass |
|---|---|---|---|
| 1 | 비로그인 + 타인 프로필 | 전시 탭만 표시, 편집/메뉴 숨김, 팔로우 시 로그인 유도 | ☐ |
| 2 | 로그인 + 타인 프로필 | 전시 탭만 + 팔로우 버튼 표시, 카드 `...` 메뉴 숨김 | ☐ |
| 3 | 로그인 + 본인 프로필 | 모든 탭(전시/작품관리/좋아요/저장/초안) + 편집 버튼 + 카드 메뉴 | ☐ |
| 4 | 로그인(강사) + 본인 | 위 + 수강생 작품 탭 추가 노출 + 프로필 헤더에 강사 배지 | ☐ |
| 5 | 타인 프로필에서 검수중 작품 | 완전히 숨겨짐 (카드 자체 미표시) | ☐ |
| 6 | 타인 프로필에서 반려 작품 | 완전히 숨겨짐 | ☐ |
